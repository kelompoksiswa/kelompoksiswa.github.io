23. Sebuah software gratis, dapat di download dan digunakan oleh pengguna. Akan tetapi penggunaan software tersebut ada batas waktunya, disebut


disebut shareware...

Pengertian Shareware adalah software gratis, dapat di download dan digunakan oleh pengguna. Akan tetapi penggunaan software tersebut ada batas waktunya, jika pengguna merasa softwarenya bagus, maka diharuskan membeli. Shareware sering dibatasi lamanya waktu pakai (misalnya trial 30 hari), atau jumlah software tersebut dijalankan (misalnya 30 kali), atau feature-feature tertentu yang tidak bisa diakses. Sesudah masa ujicobanya berakhir, software bisa saja terkunci atau bisa saja tetap berfungsi sebagaimana mestinya dan tetap terbatasi.

27. Perbedaan software ag yang gratis dan berbayar


Jawaban:

Perbedaan software yg gratis dan berbayar

- software gratis, yaitu yg berhak cipta yg gratis digunakan tanpa batasan waktu, berbeda dari shareware yang mewajibkan penggunanya membayar

software gratis (freeware) juga didefinisikan sebagai program apapun yang didistribusikan gratis, tanpa biaya tambahan.

Free Software lebih mengarah kepada bebas penggunaan tetapi tidak harus gratis. Pada kenyataannya, namanya adalah karena bebas untuk mencoba perangkat lunak sumber terbuka (Open Source) dan di sanalah letak inti dari kebebasan.

Jadi free software tidak mengarah kepada gratis pembelian tetapi penggunaan dan distribusi. Begitu keluar dari lisensi kita juga dapat menemukan berbagai cara untuk mendistribusikan perangkat lunak, termasuk freeware, shareware atau Adware.

- software berbayar, digunakan untuk tujuan komersil, jadi setiap pengguna yg ingin menggunakan / mendapatkan software tersebut dengan cara membeli / membayar pada pihak yang mendistribusikannya. Pengguna yang menggunakan software berbayar umumnya tidak diijinkan untuk menyebarluaskan software tersebut secara bebas tanpa ijin ada penerbitnya.
