18. Jika harga input tenaga kerja sebesar 3 per unit dan harga input modal sebesar 5 per unit. Agar biaya yang digunakan minimum, tentukan berapa jumlah tenaga kerja (l) dan input modal (k) yang harus di gunakan untuk memproduksi 12 unit barang ?


Pasar input adalah salah satu bentuk pasar yang menyediakan faktor-faktor produksi seperti sumber daya alam, modal, dan tenaga kerja. Berdasarkan soal, dapat disimpulkan bahwa jika harga input tenaga kerja sebesar 3 per unit dan harga input modal sebesar 5 per unit. Agar biaya yang digunakan minimum, jumlah tenaga kerja (l) dan input modal (k) yang harus digunakan untuk memproduksi 12 unit barang, jumlah dari tenaga kerja yang diperlukan adalah 12 dan input modal yang diperlukan adalah 12.

Penjelasan dengan langkah-langkah:

Diketahui:

Jika harga input tenaga kerja sebesar 3 per unit dan harga input modal sebesar 5 per unit.

Ditanyakan:

Agar biaya yang digunakan minimum, tentukan berapa jumlah tenaga kerja (l) dan input modal (k) yang harus di gunakan untuk memproduksi 12 unit barang!

Jawab:

Jumlah tenaga kerja (L) yang diperlukan untuk memproduksi 12 unit barang yaitu:

[tex]L^{\frac{3}{8} }=12K^{-\frac{5}{8}}\\((L)^{\frac{3}{8} }) ^{\frac{8}{3} }=(12K^{-\frac{5}{8} }) ^{\frac{8}{3}}\\L=12^{\frac{8}{3} }K^{-\frac{5}{8} }[/tex]

бLбK = [tex]12^{\frac{8}{3} }(-\frac{5}{3})K^{-\frac{8}{3} }[/tex]

LCC = бLбK = Pk.[tex]P^{1}[/tex]

LCC = [tex](-\frac{5}{3}).12^{\frac{8}{3} }K^{-\frac{8}{3} }[/tex]

LCC = [tex]-\frac{5}{3}[/tex]

[tex]1=12^{\frac{8}{3} }L^{\frac{8}{3} }\\L^{\frac{8}{3} }= 12^{\frac{8}{3} }\\L=12[/tex]

Input modal yang digunakan untuk memproduksi 12 unit barang dengan biaya minimum yaitu:

[tex]L^{\frac{8}{3} }K^{\frac{5}{8} }=12\\12= 12^{\frac{3}{8} }K^{\frac{5}{8} }\\12^{\frac{5}{8} }=K^{\frac{5}{8} }\\K=12[/tex]

Dengan demikian, jumlah dari tenaga kerja yang diperlukan adalah 12 dan input modal yang diperlukan adalah 12.

28. Jika diketahui harga unit input modal (K) sebesar 1500 dan harga input tenaga kerja (L) sebesar 750, anggaran yang disediakan perusahaan untuk memperoleh kombinasi input tersebut sebesar 150.000


Fungsi anggaran untuk kombinasi input sebesar 150.000 dengan harga unit input modal (K) sebesar 1500 dan harga input tenaga kerja (L) sebesar 750 adalah x + 2y = 200.

Hasil tersebut diperoleh dengan menggunakan rumus berikut:

TO = Lx + Ky

Keterangan:

TO = anggaran perusahaanL = input tenaga kerjaK = input modal

Penjelasan dengan langkah-langkah

Diketahui

Harga input tenaga kerja: L = 750Harga input modal: K = 1.500Anggaran perusahaan yang disediakan: TO = 150.000

Ditanyakan

a. Tentukan fungsi anggaran berdasarkan kombinasi input tersebut!b. Tentukan fungsi anggaran yang baru jika harga inputnya konstan tetapi anggaran perusahaannya meningkat menjadi 750.000!

Jawab

Langkah 1

Fungsi anggaran jika anggaran perusahaan yang disediakan adalah 150.000, adalah sebagai berikut:

          Lx + Ky = TO

750x + 1.500y = 150.000

    750(x + 2y) = 150.000

             x + 2y = [tex]\frac{150.000}{750}[/tex]

             x + 2y = 200

Jika x kita ganti dengan variabel w dan y diganti dengan variabel r, maka fungsi anggarannya menjadi:

w + 2r = 200

Langkah 2

Fungsi anggaran baru jika anggaran perusahaannya meningkat menjadi 75.000 dan harga inputnya konstan adalah sebagai berikut:

          Lx + Ky = TO

750x + 1.500y = 750.000

    750(x + 2y) = 750.000

             x + 2y = [tex]\frac{750.000}{750}[/tex]

             x + 2y = 1000

Jika x kita ganti dengan variabel w dan y diganti dengan variabel r, maka fungsi anggarannya menjadi:

w + 2r = 1000
