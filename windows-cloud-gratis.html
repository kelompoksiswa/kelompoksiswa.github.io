24. 4.Apakah yang Anda ketahui dengan aplikasi berbasis cloud (cloud based application) ? 5. Apakah keuntungan dari penggunaan apkikasi berbasis cloud ?


Aplikasi berbasis cloud (cloud based application) adalah aplikasi yang disediakan oleh perusahaan penyedia layanan cloud. Maksud dari layanan cloud adalah aplikasi yang kita gunakan berada di internet dan tidak terinstall di komputer kita. Untuk mengakses aplikasi tersebut kita memerlukan akses internet.

Keuntungan dari penggunaan aplikasi berbasis cloud adalah keamanan data dan kita tidak dipusingkan untuk melakukan perawatan terhadap aplikasi tersebut karena sudah menjadi tanggung jawab penyedia layanan cloud tersebut.

Pembahasan

Setelah mengetahui apa itu aplikasi berbasis cloud dan keuntungan dari menggunakan aplikasi cloud, mari kita sebutkan jenis-jenis dari aplikasi berbasi cloud, beberapa jenis aplikasi cloud adalah sebagai berikut

Infrastructure as a Service contohnya adala Amazon EC2, Telkom Cloud.Platform as a Service contoh aplikasinya adalah Windows AzureSoftware as a Service aplikasi Office 365 merupakan contoh aplikasi cloud jenis Software as a Service

Penggunaan aplikasi cloud tentu sangat membantu seorang pengusaha, karena bisa lebih fokus untuk pengembangan usahanya dan tidak dipusingkan dengan mengurus komputer dan aplikasi-aplikasi pendukungnya. Selain contoh-contoh diatas, masih ada contoh aplikasi cloud yang sering kita gunakan yaitu email dan media sosial.

29. sebutkan perbedaan penyimpanan cloud dengan non cloud​


Perbedaannya hanyalah pada bentuk fisik dan non fisik. Cloud merupakan bentuk off premise (non fisik) dari penyimpanan data melalui internet. Sedangkan data center adalah bentuk on premise (fisik) penyimpanan data pada perangkat keras.


30. 11. Komponen dari dua data atau lebih infrastruktur cloud disebut... a. Hybrid cloud b. Community cloud c. On-site-cloud d. Private cloud e. Public cloud​


Jawaban:

A. Hybrid cloud

Penjelasan:

Hybrid cloud adalah jenis infrastruktur cloud yang terdiri dari dua atau lebih komponen dari jenis infrastruktur cloud yang berbeda, seperti public cloud, private cloud, atau on-premises cloud. Dalam hybrid cloud, setiap jenis infrastruktur cloud tetap terpisah, tetapi dihubungkan melalui jaringan yang aman dan dapat berkomunikasi dengan lancar satu sama lain. Dengan menggunakan hybrid cloud, organisasi dapat memanfaatkan keuntungan dari setiap jenis infrastruktur cloud yang tersedia, misalnya keamanan dan kontrol yang lebih baik dari private cloud dan skalabilitas dan elastisitas dari public cloud. Hybrid cloud juga memberikan fleksibilitas bagi organisasi untuk memilih infrastruktur cloud yang paling cocok dengan kebutuhan bisnis mereka serta memungkinkan organisasi untuk memindahkan data dan aplikasi dari satu infrastruktur cloud ke infrastruktur cloud lainnya jika diperlukan.
