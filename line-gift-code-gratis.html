9. Bagaimana membuat agar kita bisa jump ke line tertentu sehingga fungsi dibawah line tersebut tidak dieksekusi melainkan jump ke line tertentu (C++)?


Jawaban:

Gunakan statement goto.

Penjelasan:

Statement goto digunakan untuk membuat jump dari line tertentu ke line yang sudah dilabeli tanpa menggunakan kondisi apapun seperti if statement. Contohnya adalah sebagai berikut

#include <iostream>

using namespace std;

void uselessFunction()

{

   cout << "Useless" << endl;

}

int main()

{

   cout << "Next Statement is skipped" << endl;

   goto notUseless;

   uselessFunction();

   notUseless: cout << "Previous Statement is skipped" << endl;

   return 0;

}

Program ini akan menjalankan statement cetak "Next Statement is skipped" kemudian ke statement goto notUseless dimana label notUseless ada di bawah uselessFunction(); yang menyebabkan uselessFunction tidak berjalan dan mencetak "Previous Statement is skipped".

14. Bagaimana Caranya Untuk Masuk Ke Grup Line Tanpa Diinvite(Menggunakan Qr code Dan Link).Tolong Dibantu Ya!


Untuk masuk ke grup Line tanpa diundang atau di-invite, bisa dengan menggunakan QR Code ataupun Link. Kedua cara baru bisa terlaksana apabila pilihan invite by link – QR Code sudah tercentang di bagian pengaturan obrolan grup Line. Pengaturan ini bisa diakses pada gambar gear yang ada di kanan atas layar sejajar dengan menu posts, album dan member.

Setelah opsi invite by link – QR Code selesai dicentang, kembali lah ke halaman utama dari grup lain dan pilih opsi Add+ atau Undang. Setelahnya akan muncul opsi untuk mengundang member grup line lewat QR – Code, Link, E-mail dan SMS.

Apabila yang dipilih adalah opsi LINK, maka akan muncul tautan untuk dicopy yang bisa diberikan pada teman yang hendak menjadi member grup. Apabila memilih opsi QR – Code maka akan muncul gambar kode (barcode) yang bisa dikirimkan ke teman yang hendak bergabung. Untuk membaca barcode tersebut, teman kamu sebaiknya menginstal barcode scanner terlebih dahulu di playstore.
