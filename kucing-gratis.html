1. - ALJABAR SERIES -3x(y² + xy)(x - 3)²Note : Pasangan Kucing tadi ( ambil aja gratis )​


[tex] \mathfrak{ \huge{ \red{ aljabar}}}[/tex]

_____________

» Soal

3x(y² + xy)

(x - 3)²

______________

» Penyelesaian

3x( y² + xy )

= ( 3x )( y² ) + ( 3x )( xy )

= 3xy² + ( 3x )( xy )

= 3xy² + 3x²y

-----

( x - 3 )²

= x² - (2)(x)(3) - 3²

= x² - 6x - 9

atau....

( x - 3 )²

= ( x - 3 )( x - 3 )

= x² - 3x - 3x + 9

= x² - 6x + 9

>>ALJABAR SERIES___________

( x - 3 )²

= x² - (2)(x)(3) - 3²

= x² - 6x - 9

__________

( x - 3 )²

= ( x - 3 )( x - 3 )

= x² - 3x - 3x + 9

= x² - 6x + 9
