4. apa perbedaan web gratis dengan web berbayar


Jawaban:

Perbedaan web gratis dan berbayar yang pertama ada pada server yang digunakan. Bagi pengguna web gratis, mereka biasanya tidak memiliki pengetahuan tentang letak server ataupun kualitas serta teknologi yang digunakan. Jika situs yang anda miliki lebih bersifat personal, hal ini tidak akan menjadi masalah namun jika web anda digunakan untuk tujuan bisnis, anda wajib mengetahui seluk beluk server yang ditawarkan. Penyedia jasa hosting berbayar akan memberikan anda informasi tentang hal ini sehingga anda bisa memilih sendiri server yang paling tepat untuk bisnis khususnya jika berkaitan dengan lokasi penempatan server.
