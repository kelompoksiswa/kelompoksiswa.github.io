18. mohon bantuannya The picture is a graph of quadratic function......​


Persamaan umum fungsi kuadrat :

f(x) = Ax² + Bx + C

Diketahui grafik fungsi f(x) melewati titik :

» (3 , 0)

=> f(3) = 0

=> A.(3)² + B.(3) + C = 0

=> 9A + 3B + C = 0 ( i )

» (–1 , 0)

=> f(–1) = 0

=> A.(–1)² + B.(–1) + C = 0

=> A - B + C = 0 ( ii )

» (0 , 3)

=> f(0) = 3

=> A.(0)² + B.(0) + C = 3

=> C = 3

Subtitusikan nilai C ke ( i ) :

9A + 3B + C = 0

9A + 3B + 3 = 0

9A + 3B = –3 ( iii )

Subtitusikan nilai C ke ( ii ) :

A - B + C = 0

A - B + 3 = 0

A - B = –3 ( iv )

Eliminasi ( iii ) dan ( iv ) :

( iii ) : 9A + 3B = –3

(iv) × 3 : 3A - 3B = –9

________________ +

12A = –12 => A = –1

Subtitusikan nilai A ke ( iv ) :

A - B = –3

–1 - B = –3 => B = 2

Substitusikan nilai A, B, dan C ke persamaan fungsi kuadrat :

f(x) = Ax² + Bx + C

f(x) = (–1)x² + (2)x + (3)

Jadi, persamaan fungsi kuadrat grafik tersebut adalah : f(x) = –x² + 2x + 3


19. Gambarkan sebuah graph dengan 7 buah simpul, masing-masing derajatnya adalah 2,2,4,4,4,2,4 dan tentukan jenis graph apa ?​


Jawab:

Untuk menggambarkan sebuah graph dengan 7 buah simpul, masing-masing derajatnya adalah 2,2,4,4,4,2,4, maka kita dapat menggambarkannya seperti berikut:

Graph:

2

|

2 4

| |

4 4 2

| | |

4 4 4

Dari gambar di atas, dapat dilihat bahwa graph tersebut merupakan sebuah graph terhubung dengan 7 buah simpul. Masing-masing simpul memiliki derajat 2, 2, 4, 4, 4, 2, 4. Jenis graph yang terbentuk dari data tersebut adalah sebuah graph yang disebut graph reguler dengan derajat 4, karena semua simpul memiliki derajat yang sama yaitu 4.

Sebagai informasi tambahan, graph reguler adalah sebuah graph yang memiliki simpul-simpul yang semuanya memiliki derajat yang sama. Graph reguler dapat dibagi menjadi beberapa jenis, yaitu graph reguler dengan derajat 2 (disebut dengan graph cincin), graph reguler dengan derajat 3 (disebut dengan graph segitiga), dan graph reguler dengan derajat 4 (disebut dengan graph kubus).
