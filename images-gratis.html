[tex]= \int\ {(x^3)(x^4-3)^{-6}} } \, dx[/tex]
[tex]= \int\ {(x^3)u^{-6}} } \, \frac{du}{4x^3} [/tex]
[tex]= \frac{1}{4} \int\ u^{-6}} } \, du[/tex]
[tex]=(\frac{1}{4}) (\frac{1}{-5})u^{-5}} } +c[/tex]
[tex]=-\frac{1}{20}u^{-5}} } +c[/tex]

[tex]=- \frac{1}{20(x^4-3)^{5}}+c [/tex]

Untuk sebarang bilangan bulat p dan bilangan bulat positif berlaku

[tex]P^{n}[/tex] = {p × p × p × p × ….. × p} sebanyak n faktor

[tex]P^{n}[/tex] disebut bilangan berpangkat, p disebut bilangan pokok, dan n disebut pangkat (eksponen).
