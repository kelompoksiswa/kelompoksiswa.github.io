1. Cara memperlancar browsing


1). Hindari Jam Sibuk
Jangan melakukan browsing pada jam-jam sibuk (09.00-17.00) agar browsermu tidak "Terjebak Kemacetan" lalu lintas di dunia maya. Pada jam-jam itulah banyak orang yang mengakses Internet. 2). Tidak Menampilkan Animasi dan Multimedia
Animasi memang menjadi hal yang menyenangkan , tetapi memperberat kerja browser. Jadi sebaiknya jika kalian ingin mempercepat/memperlancar Browsing anda nonatifkanlah animasinya. 3). Tambahkan Cache Browser
Pada waktu kalian mengunjungi sebuah situs, Internet Explorer menempatkan kode HTML dan grafik dari situs tersebut pada cache. Cache membantu Internet Explorer bekerja lebih cepat pada saat kamu mengklik tombol Back 4). Hentikan Proses Download Yang Terlalu Lama
Jika Browser terlalu lama mengakses suatu halaman web dan belum selesai juga, sebaiknya kamu klik tombol Stop. Terkadang, rute yang digunakan browser macet. Dengan mengulang proses download, browser dapat menemukan jalur baru yang agak kosong di Intenet. 5). Menyimpan Situs Favorit Kalian
Jika kamu menemukan sebuah situs/halaman web yang menarik dan berencana untuk mengunjunginya kembali pada suatu waktu, sebaiknya kamu memasukkanya ke dalam daftar favorit. 6). Offline Browsing
Offline Browsing adalah sebuah cara untuk menghemat penggunaan waktu online. Dengan offline Browsing, kamu dapat menyimpan sebuah dokumen web ketika sedang online agar dapat dibuka kembali kapan saja meskipun dalam keadaan offline

Bagaimanakah cara menyimpan suatu halaman agar dapat di buka kembali dengan offline browsing
Untuk menyimpan halaman web, ada 2 cara:

Cara pertama:
1. Klik kanan pada web yang sedang terbuka
2. Klik Save As
3. Pilih lokasi penyimpanan
4. Halaman web akan tersimpan dengan format .html

Cara Kedua:
1. Klik menu pada browser
2. Klik Save Page As
3. Pilih lokasi penyimpanan
4. Halaman web akan tersimpan dengan format .html

• Penjelasan:

Untuk membuka halaman web yang telah disimpan, cukup klik 2 kai (double klik) pada file halaman website .html yang telah tersimpan. Dengan cara Save Page (Ctrl+S), atau dengan cara Menu >> Save Page as >> Simpan dalam format .html
