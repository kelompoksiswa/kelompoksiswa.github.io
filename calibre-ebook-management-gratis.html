Sebutkan dan jelaskan 15 format ebook yg didukung oleh calibre

Format buku yang didukung oleh Ebook Readr Calibre antara lain:

1. EPUB
Epub adalah salah satu format untuk aplikasi e-book yang bersifat terbuka, keistimewaan format epub dibandingkan dengan yang lain adalah kemampuan untuk melakukan optimasi tampilan text menyesuaikan diri dengan device yang dipakai untuk membuka ebook.

2. MOBI
Mobipocket e-book format berdasarkan ebook Terbuka standar menggunakan XHTML dan dapat mencakup JavaScript dan bingkai.

3. AZW
Sebuah format proprietary Amazon. Hal ini menyerupai format MOBI kadang-kadang dengan dan kadang-kadang tanpa menyertakan Digital Rights Management (DRM). DRM pada format ini dikhususkan untuk Kindle Amazon.
4. PRC
Palm Resource File. Sering menyertakan e-book Mobipocket tapi kadang-kadang menyertakan eReader atau e-book AportisDoc.
5. LRF
Format buku digital yang digunakan oleh Sony Corporation . Ini adalah format proprietary, tetapi beberapa perangkat lunak pembaca untuk-tujuan komputer umum, terutama di Linux (misalnya, internal pemirsa kaliber), memiliki kemampuan untuk membacanya. Ekstensi file lrx merupakan e-book terenkripsi DRM.

6. PDB
Software TealPoint milik pembaca untuk Palm OS. Selain format sendiri, ini akan membuka file teks biasa dan PalmDoc.

7. CBR
Berkas Comic Book Archive file atau ComicBook Reader terdiri dari serangkaian file gambar, biasanya PNG (kompresi lossless) atau JPEG (kompresi lossy) file, disimpan sebagai file arsip tunggal, untuk tujuan melihat urutan gambar, buku komik terutama.

8. CHM
Compressed HTML, sering digunakan untuk file bantuan Windows. Hal ini telah menjadi sangat populer untuk distribusi teks dan bahan pendukung lainnya melalui Web.

9. DJVU
Format djvu (dibaca DeJaVu) adalah format file komputer yang dirancang untuk menyimpan dokumen hasil scan, terutama yang berisi kombinasi text, gambar garis dan foto.

10. FB2
FictionBook adalah populer XML-based e-book format, didukung oleh pembaca gratis seperti FBReader , Haali Reader, STDU Viewer, dan Calibre.

11. HTML
Hyper Text Markup Language adalah tulang punggung dari World Wide Web. Banyak teks yang didistribusikan dalam format ini. Selain itu, beberapa pembaca e-book mendukung Cascading Style Sheets (CSS) yang pada dasarnya gayautama panduan  untuk halaman HTML.

12. LIT
Format LIT merupakan format dari Microsoft Reader yang memungkinkan teks dalam buku elektronik disesuaikan dengan lebar layar divais mobil yang digunakan untuk mebacanya.

13. PDF
Portable Document Format yang diciptakan oleh Adobe untuk produk Acrobat mereka. Format ini secara tidak langsung merupakan format yang digunakan untuk pertukaran dokumen.

14. ODT
Open Document Format atau dikenal juga sebagai Open Document atau ODF saja adalah dokumen berformat terbuka. Merupakan file berformat XML yang digunakan sebagai standar format file terbuka pada aplikasi-aplikasi Perkantoran (Office Suite).

15. TXT
.txt merupakan format standar yang dipakai untuk menyimpan seluruh teks yang belum diformat. Format ini biasanya dikenal oleh sebagian besar perangkat lunak.

2. Jelaskan cara melakukan upgrade informasi metadata ebook yang dapat anda buat dengan Calibre​


Tahapan memodifikasi informasi metadata sebuah ebook menggunakan aplikasi calibre adalah sebagai berikut

Untuk memodifikasi informasi metadata satu per satu, langkah-langkahnya adalah sebagai berikut

- Klik ebook yang akan dimodifikasi

- Klik tombol Edit Metadata atau tekan tombol E. Akan muncul kotak dialog yang memungkinkan kita untuk memodifikasi semua aspek dari metadata

- Untuk menukar informasi judul dan pengarang secara otomatis, klik tombol diantara judul dan pengarang

- Tekan tombol disamping Author Sort untuk mengisi nilai secara otomatis. Gunakan Manage Author untuk melihat dan mengubah nilai sortir.  

- Klik tombol disamping Tag untuk menggunakan Tag Editor untuk mengatur semua tag yang terasosiasi dengan ebook

- Kotak Ids dapat digunakan untuk memasukkan ISBN (dan jenis ID lainnya).    Kotak ini akan berwarna merah jika ISBN yang dimasukkan tidak valid. Jika ISBN valid, maka kotak akan berwarna hijau

- Kotak Author Sort akan berwarna merah jika nilai sortasi berbeda dengan kaliber

Untuk memodifikasi informasi metadata semua ebook, langkah-langkahnya adalah sebagai berikut

- Pilih semua ebook yang akan dimodifikasi degan menekan tombol Ctrl atau Shift dan meng-klik ebook tersebut.  

- Klik tombol Edit Metadata. Akan muncul kotak dialog yang memungkinkan kita untuk memodifikasi semua aspek dari metadata

- Tentukan isi dari Author, Publisher, Rating, Tags, Series, dan lain-lain. Nilai akan sama untuk semua ebook. Modifikasi ini dapat dilakukan jika semua ebook memiliki beberapa kesamaan dalam metadata

13. calibre adalah contoh dari​


Jawaban:

Tujuan

penjelasan:

Calibre adalah program yang dikhususkan untuk manajemen ebook di komputer. Seperti perpustakaan, kita dapat mengurutkan ebook menurut judul, pengarang, ukuran, tanggal, dan lain-lain sehingga memudahkan dalam pencarian. Kita juga dapat menambahkan kategori dan komentar untuk ebook. Selain itu Calibre dapat menemukan metadata ebook berdasarkan judul, pengarang, atau informasi ISBN lewat koneksi internet.

Dengan Calibre kita juga bisa mengkonversi format ebook supaya dapat dibaca oleh piranti-piranti elektronik yang kadang hanya terbatas pada format tertentu saja.

Format yang didukung adalah :

Input : CBZ, CBR, CBC, CHM, EPUB, FB2, HTML, LIT, LRF, MOBI, ODT, PDF, PRC**, PDB, PML, RB, RTF, SNB, TCR, TXT.

Output : EPUB, FB2, OEB, LIT, LRF, MOBI, PDB, PML, RB,

PDF, SNB, TCR, TXT.

Kelebihan lain dari Calibre adalah mengambil berita dari website atau RSS feed untuk dijadikan ebook. Contoh situs berita yang didukung Calibre adalah Kompas, Jakarta Post, New York Times, ESPN, The Guardian, The Wall Street Journal, Times, dan lain sebagainya. Biasanya akan dikategorikan sesuai dengan negaranya.

18. Tuliskan cara membuat cover pada software calibre!


Jawaban:

Langkah pertama, Instalasi Calibre ke dalam perangkat yang kita gunakan (PC/Laptop/Mobile) :

1. Installah software Calibre yang sudah anda download, dengan mendouble klik file calibre-0.9.10.msi. Maka akan tampil :

Pilih bahasa yang akan digunakan, dan klik next.

2. Pilih prangkat ebook yang digunakan, misalnya Android dan pilih jenisnya. Selanjutnya klik Next. Maka akan tampil :

 

3. Klik Finish.

Jika Calibre sudah terinstal dengan baik, jalankan Calibre dengan mendouble klik icon sehingga tampil :

Mengkonversi dokumen (.doc) ke dalam format HTML

1. Bukalah dokumen anda yang tersimpan dalam format .doc/docx

2. Lakukan pengeditan yang diperlukan sebelum melakukan konversi ke dalam format HTML. Sebaiknya anda mengedit/membuang karakter-karakter yang tidak diperlukan (seperti left indent/enter yang berlebihan). Pastikan dokumen terlihat padat dan tidak ada space (ruang kosong) yang tidak diperlukan.

3. Klik File, Save As, pada Save as type pilih Web Page Filtered.

4. Klik Save.

Selanjutnya, adalah Membuat ebook dengan Calibre :

1. Jalankan Calibre dengan mendouble klik icon  

2. Klik Add books > Add books from a single directory > Pilih file HTML hasil konversi dari file .doc sebelumnya. Tunggulah sampai proses adding/upload selesai dan file terlihat pada daftar file yang akan di-convert.

3. Selanjutnya, klik daftar file yang akan di-convert menjadi ebook (epub) dan klik icon Convert books > Convert individually. Maka akan muncul kotak dialog Convert….

Pada Output format : Pilih EPUB (format default Calibre)

Change cover imager : Pilih gambar kover ebook dan beri tanda centang pada use cover from source file

Lengkapi Metadata Title, Author, Author sort, Publisher, Tags dan series.

4. Klik OK. Dan tunggulah… Calibre sedang mengkonversi ke format EPUB

5. Jika sudah selesai, untuk melihat hasilnya, klik file yang ada pada daftar dan klik icon View. Maka hasil ebook dengan format EPUB akan ditampilkan.

Note :

File ebook (EPUB) hasil konversi secara default akan disimpan di C:\My Documents\Calibre Library\ berupa sebuah folder berisi file EPUB dan beberapa file pendukung lainnya.

Untuk menjalankannya/melihat hasil ebook, anda bisa membukanya menggunakan ebook viewer (bawaan Calibre) dengan cara mengeklik kanan file berformat EPUB > Open with > Pilih Ebook viewer (pada C:\program files\calibre2\)

Selamat mencoba, mudah-mudahan bermanfaat… ;)
