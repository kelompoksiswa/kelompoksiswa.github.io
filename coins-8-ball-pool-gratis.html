25. christy has several $5 coins and $2 coins. it is given the total value of the coins is $38. if she uses two $5 coins to buy a pen, then the numbers of $5 coins and $2 coins will be the same. find the original number of coins


Jawaban:

The original number of $5 coins is 6.The original number of $2 coins is 4.

Penjelasan dengan langkah-langkah:

Number of $5 coins = x

Number of $2 coins = y

Total value is $38:

5x + 2y = 38 ... (1)

If two $5 coins used, then the number of $5 and $2 coins will be the same:

y = x - 2 ... (2)

Substitute the value of y to equation (1):

5x + 2y = 38

5x + 2(x-2) = 38

5x + 2x - 4 = 38

7x - 4 = 38

7x = 38 + 4

7x = 42

x = 42/7

x = 6

y = x-2

y = 6-2

y = 4

Therefore:

The original number of $5 coins is 6.The original number of $2 coins is 4.

30. Jo , Kelly and Linda had 5102 coins altogether . Linda had 3 times as many coins as Kelly. Jo had 362 more coins than Kelly . How many coins did Jo have?


1310 Coins

Penjelasan dengan langkah-langkah:

From what we have, we will say that Kelly has an x amount of coins, that means Linda has 3x coins. and Jo has x + 362. Let's list that down.

Jo: x + 362

Linda: 3x

Kelly: x

we can work from that, that means if we add it all together that has a result of:

5x + 362 = 5102

Now, we move the total around,

5102 = 362 + 5x

5102 - 362 = 4740

4740 = 5x

5x = 4740

4740 ÷ 5 = 948

x = 948

Now we know how much Kelly has, now we add it up.

x + 362 = 1310

So,

Jo has 1310 Coins.
