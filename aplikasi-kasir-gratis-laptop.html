15. Sebuah laptop memiliki tenaga cukup untuk menyala tanpa aplikasi selama 6 jam. Di dalamnya ada sebuah aplikasi yang jika dinyalakan membuat tenaga laptop hanya bertahan selama 4 jam. Jika dari tenaga penuh laptop itu menyala dengan aplikasi selama 40 menit sebelum kemudian aplikasi itu dimatikan, berapa sisa waktu agar tenaga laptop itu habis?


Jawaban:

300 menit/5 jam

Penjelasan:

Misal tenaga laptop x

Diketahui

Kecepatan tenaga laptop habis tanpa aplikasi x/6 jam atau x/360 menit

Kecepatan tenaga laptop habis dengan aplikasi x/4 jam atau x/240 menit

Laptop dipakai dengan aplikasi selama 40 menit maka sisa tenaga

x - x/240 * 40 = x - x/6

Lalu digunakan tanpa apliaksi maka waktu sampai habis

t = sisa tenaga / kecepatan habis

t = x-x/6 / x/360

t = x- x/6 * 360/x

t = 360-60=300 menit

Bagaimana cara menginstall aplikasi pada laptop/komputer?

Jika telah memiliki file instalasi aplikasinya,

1. Buka File Explorer

2. Buka folder dimana file instalasi aplikasi berada (file umunya berekstensi .exe atau .msi)

3. Klik 2x (Double klik) pada file tersebut

4. Baca dan ikuti instruksi pada dialog instalasi yang muncul (misalnya setujui EULA, klik tombol Next, dll.)

5. Tunggu proses intalasi selesai, dan klik tombol Finish

6. Aplikasi telah terinstall

• Penjelas:

Untuk membuka aplikasi yang telah terinstall, dapat klik 2x pada shortcut aplikasi yang ada di Dekstop, atau dengan Klik tombol start > All program > Klik aplikasi yang telah diinstall.

3. ketika ada pop up berupa file aplikasinya dari browser, klik download

9. ketika sudah, klik next. ketika ada bar loading, penginstalan sedang berjalan. tunggu hingga selesai
10. ketika sudah selesai, klik finish. ketika ada pilihan "run this program after finish the installation", klik saja jika kamu ingin menjalankan program secara langsung
