6. Jelaskan cara melakukan upgrade informasi metadata ebook yang dapat anda buat dengan Calibre​


Tahapan memodifikasi informasi metadata sebuah ebook menggunakan aplikasi calibre adalah sebagai berikut

Untuk memodifikasi informasi metadata satu per satu, langkah-langkahnya adalah sebagai berikut

- Klik ebook yang akan dimodifikasi

- Klik tombol Edit Metadata atau tekan tombol E. Akan muncul kotak dialog yang memungkinkan kita untuk memodifikasi semua aspek dari metadata

- Untuk menukar informasi judul dan pengarang secara otomatis, klik tombol diantara judul dan pengarang

- Tekan tombol disamping Author Sort untuk mengisi nilai secara otomatis. Gunakan Manage Author untuk melihat dan mengubah nilai sortir.  

- Klik tombol disamping Tag untuk menggunakan Tag Editor untuk mengatur semua tag yang terasosiasi dengan ebook

- Kotak Ids dapat digunakan untuk memasukkan ISBN (dan jenis ID lainnya).    Kotak ini akan berwarna merah jika ISBN yang dimasukkan tidak valid. Jika ISBN valid, maka kotak akan berwarna hijau

- Kotak Author Sort akan berwarna merah jika nilai sortasi berbeda dengan kaliber

Untuk memodifikasi informasi metadata semua ebook, langkah-langkahnya adalah sebagai berikut

- Pilih semua ebook yang akan dimodifikasi degan menekan tombol Ctrl atau Shift dan meng-klik ebook tersebut.  

- Klik tombol Edit Metadata. Akan muncul kotak dialog yang memungkinkan kita untuk memodifikasi semua aspek dari metadata

- Tentukan isi dari Author, Publisher, Rating, Tags, Series, dan lain-lain. Nilai akan sama untuk semua ebook. Modifikasi ini dapat dilakukan jika semua ebook memiliki beberapa kesamaan dalam metadata

23. Jelaskan cara melakukan upgrade informasi metadata ebook yang anda buat dengan calibri?.


Untuk melakukan upgrade informasi metadata ebook yang Anda buat dengan Calibre, pertama-tama Anda perlu membuka aplikasi Calibre dan menemukan ebook yang ingin Anda upgrade metadata-nya. Kemudian, ikuti langkah-langkah berikut:

Klik kanan pada ebook yang ingin Anda upgrade metadata-nya, kemudian pilih "Edit metadata" dari menu yang muncul.Pada jendela yang muncul, Anda dapat mengubah informasi metadata ebook tersebut sesuai keinginan Anda. Informasi metadata yang dapat diubah antara lain judul, penulis, deskripsi, kategori, dan lain-lain.Setelah selesai mengubah informasi metadata, klik tombol "OK" untuk menerapkan perubahan.Jika Anda ingin menyimpan perubahan tersebut secara permanen, Anda juga perlu mengklik tombol "Save" di bagian atas jendela.

Dengan demikian, informasi metadata ebook yang Anda buat dengan Calibre akan ter-upgrade sesuai dengan perubahan yang Anda lakukan.
