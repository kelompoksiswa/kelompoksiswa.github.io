Jawaban :

200 m

Penjelasan : → Diketahui : Jarak pada peta ( JP ) = 2 cm Skala ( S ) = 1 : 10.000

→ Ditanya : Jarak Sebenarnya ( JS ) = ???

→ Penyelesaian :

JS = JP × S

= 2 × 10.000

= 20.000 cm

Ubah ke satuan m

JS = 20.000 × 1/100

= 20.000 / 100

= 200 m

Jadi, jarak sebenarnya dari Alfamart A ke Alfamidi B adalah 200 m
