4. prosedur untuk memasang aplikasi(perangkat lunak) keprogrqm so windows xp


1. buka program installer untuk menginstal program dengan 2 kali klik
2. ketika sudah muncul jendela penginstalan (ucapan selamat datang), klik next
3. ketika program yang diinstal tersebut meminta persetujuan berupa EULA, baca persetujuannya, kalau setuju, klik pada tanda centang "aku setuju dengan persetujuannya" lalu klik next
4. ketika program yang diinstal akan memperlihatkan paket yang ingin diinstal, pilih paket yang akan diinstal (diperlukan saja) atau pilih semua paket (kalau mau)
5. ketika jendela penginstalan menampilkan tempat menyimpan file program, pilih tempat menyimpan filenya lalu klik next
6. ketika semua yang diatas sudah dilakukan, tunggu hingga instalasi selesai
7. ketika sudah, klik finish

25. langkah langkah untuk mengaktifkan program aplikasi Microsoft office word 2010 dari desktop windows XP adalah​


Misalkan memakai sistem operasi Windows XP:

1. Langkah pertama, klik Start

2. Langkah kedua, klik All Programs

3. Langkah ketiga, klik Microsoft Office

4. Langkah keempat, klik Microsoft Office Word 2010

Setelah langkah keempat dilakukan, maka akan tampil program Ms. Word 2010 yang menandakan bahwa program itu telah aktif.

Stand by, jika menginginkan komputer tetap dalam keadaan siap menerima perintah atau masukan setelah komputer dimatikan.5.
