Jawaban:

Feed adalah deretan konten yang dapat Anda scroll. Konten muncul berulang satu demi satu di blok yang terlihat serupa. Contohnya, feed dapat berupa editorial (misalnya, daftar artikel atau berita) atau listingan (misalnya, daftar produk, layanan, dll.). Feed dapat muncul di mana saja di halaman. Catatan: Untuk iklan In-feed, feed tidak mengacu ke feed distribusi web seperti RSS.

Beberapa contoh feed mencakup:

Feed utama di halaman beranda (misalnya, feed berita)

Feed utama di halaman hasil produk atau layanan

Feed horizontal di halaman

Feed yang hanya berisi teks

Pembahasan Present Continuous

Digunakan untuk menyatakan Kegiatan yang sedang berlangsung saat ini atau sedang terjadi sekarang.

Rumus Present Continuous (+) S + Be (is/am/are) + V-ing + O(-) S + Be (is/am/are) + not + V-ing + O(?) Be (is/am/are) + S + V-ing + O

Catatan Be “am” untuk Subjek “I”Be “is” untuk Subjek “He/She/it/satu nama”Be “are” untuk Subjek “You/we/they/Dua nama”

Kesimpulan Subject => Jack (satu nama)Be => is (karena Subjek satu nama)Verb-ing => feeding Object => His bird
