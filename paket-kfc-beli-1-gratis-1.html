20. Pada sebuah gerai makanan cepat saji, Anto membeli 2 paket A dan 3 paket B seharga 66.000 rupiah, sedangkan Broto membeli 3 paket A dan 2 paket B seharga 69.000 rupiah. Jika Karto membeli 1 paket A dan 1 paket B maka dia harus membayar sebesar ... rupiah.​


Jawab:

Penjelasan dengan langkah-langkah:

misal : paket a = a

           paket b = b

2a + 3b = 66.000 ... (1)

3a + 2b = 69.000 ... (2)

a + b = ...?

eliminasi a pada pers. (1) dan (2)

2a + 3b = 66.000     |x3|     6a + 9b = 198.000

3a + 2b = 69.000     |x2|     6a + 4b = 138.000 -

                                                    5b = 60.000

                                                       b = 60.000 : 5

                                                       b = 12.000

substitusi nilai b ke pers. (1) atau (2)

3a + 2b = 69.000

3a + 2(12.000) = 69.000

3a + 24.000 = 69.000

3a = 69.000 - 24.000

3a = 45.000

 a = 45.000 : 3

 a = 15.000

substitusi nilai a dan b ke pers. yg ditanyakan

a + b = 15.000 + 12.000 = 27.000

jadi, yang harus dibayar karto adalah Rp. 27.000.
