-buka rufus trs open iso windows 10 yg di download tdi

-tekan start, tunggu ampe selesai

lanjut ke install

-restart laptop (posisi flashdisk tersambung)

-buka bios(klik f2 biasanya)

-dibagian device naikkan flashdisk ke no 1 (tekan f5/f6 atau - +)

-lalu save (f10)

-stelah muncuk logo windows klik start

-next aj trs smpe ke tahap install(pilih costum instalasi)

-dibagian ini atur pastisi jgn smpe salah, hapus saja partisi yg C dengn System

-lalu klik new partisi, apply

-tinggl next tunggu ampe restart

-setelah restart cabut flasdisk/kembali ke bios(cara ribet)

finishing

-disini tinggl next aj

-isi nama, password, tanggl

-selesai:)
