3. 1. Jelaskan dua cara mengupload plugin ke wordpress secara offline! 2. Jelaskan bagaimana proses mengubah wordpress agar bisa menjadi sebuah toko online dengan menggunakan plugin!


Jawaban:

Cara mengupload plugin ke WordPress secara offline: a. Melalui FTP:

•Unduh plugin yang ingin Anda upload ke situs WordPress Anda

•Ekstrak file plugin tersebut

•Buka program FTP (File Transfer Protocol) seperti FileZilla

•Masukkan informasi akun FTP untuk menghubungkan dengan situs WordPress Anda

•Cari folder "wp-content/plugins" di dalam folder WordPress Anda

•Salin folder plugin yang diekstrak ke dalam folder "plugins" di WordPress

b. Melalui cPanel:

•Unduh plugin yang ingin Anda upload ke situs WordPress Anda

•Ekstrak file plugin tersebut

•Login ke akun cPanel Anda

•Cari opsi "File Manager"Buka folder "public_html" dan temukan folder "wp-content/plugins"

•Klik tombol "Upload" di bagian atas dan pilih file plugin yang sudah diekstrak

•Tunggu hingga proses upload selesai

Proses mengubah WordPress agar bisa menjadi sebuah toko online dengan menggunakan plugin: a. Instal plugin eCommerce seperti WooCommerce

•Login ke dashboard WordPress Anda

•Klik opsi "Plugins" di sidebar kiri

•Klik tombol "Add New" di bagian atas

•Cari plugin eCommerce seperti WooCommerce

•Klik tombol "Install Now" dan tunggu hingga proses selesai

•Aktifkan plugin tersebut

b. Konfigurasi plugin WooCommerce

•Setelah plugin WooCommerce diinstal, Anda akan diarahkan ke halaman konfigurasi

•Ikuti petunjuk pada halaman tersebut untuk mengkonfigurasi toko online Anda

•Atur pengaturan seperti alamat toko, jenis mata uang, dan metode pengiriman

•Atur produk dan kategori produk yang akan dijual di toko online Anda

c. Tambahkan fitur lain yang dibutuhkan untuk toko online

•Instal plugin tambahan untuk menambahkan fitur seperti pembayaran online, pengiriman internasional, atau integrasi dengan media sosial

•Konfigurasikan plugin tambahan tersebut untuk memastikan toko online Anda berjalan lancar

d. Buat halaman toko onlineBuat halaman baru di WordPress dan beri nama "Toko" atau "Belanja"

•Tambahkan shortcode WooCommerce ke halaman tersebut

•Simpan dan terbitkan halaman baru tersebut

e. Mulai menjual produk di toko online Anda

Tambahkan produk baru ke toko online Anda

Atur harga, deskripsi, dan gambar produk

•Tambahkan produk ke kategori yang sesuai

Mulai promosikan toko online Anda dan produk yang dijual ke pelanggan potensial.
