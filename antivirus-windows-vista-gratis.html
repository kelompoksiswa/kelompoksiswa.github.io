12. Apakah fungsi Windows XP,Windows Me,dan Windows Vista????


Windows XP, Windows Me, dan Windows Vista adalah suatu OS (Operating System) yg terdapat di dalam sebuah komputer. OS berguna sebagai suatu kesatuan sistem yg terdiri dari berbagai proses dimana terdapat input-proses-output dalam sebuah komputer. Sebuah komputer tidak akan dapat difungsikan jika tidak terdapat OS terinstal didalamnya.
