Penjelasan dengan langkah-langkah:

Faiz : 7 hari sekali

Moses : 5 hari sekali

Faktor

7 = 1 x 7

5 = 1 x 5

KPK = 5 x 7 = 35

mereka bertemu lagi pada

28 September 2020 + 35 Hari

= senin, 02 November 2020

Jawaban:

2 November 2020

Penjelasan dengan langkah langkah:

Seminggu ada 7 hari

Faiz= 7 hari sekali

Moses= 5 hari sekali

Faktorisasi Prima= FP

FP dari 7= 1 × 7

FP dari 5= 1 × 5

KPK dari 7 dan 5= 1 × 7 × 5= 35

Pergi bersama lagi= 28 September 2020 + 35= 63

Karena dibulan september hanya ada 30 hari maka mereka pergi bersama pada= 63 - 30= 33 Oktober 2020

Tapi bulan Oktober sampai tanggal 31 maka= 33 - 31= 2 November 2020

26. Kuis Misal saya ada 10⁶ kupon diskon 99% di toko Henri Store. Apakah barang seharga Rp10.000,00 menjadi gratis jika menggunakan seluruh kupon? Benar/SalahJelaskan jawaban Anda. ​


Jawaban:

SALAH

Penjelasan dengan langkah-langkah:

Ada 10⁶ kupon diskon 99%

Persentase Harga setelah menggunakan 1 kupon = 100%-99% = 1%

persentase harga setelah menggunakan 10⁶ kupon

= [tex]{(1\%)}^{ {10}^{6} } [/tex]

= [tex] (\frac{1}{ {10}^{2} } ) {}^{ {10}^{6} } [/tex]

= [tex] \frac{1}{ {10}^{2 \times {10}^{6} } } [/tex]

= [tex] \frac{1}{ {10}^{2.000.000 } } [/tex]

= [tex] \frac{1}{ {10}^{1.999.998 } }\% [/tex]

karena harga tidak mencapai 0% melainkan [tex] \frac{1}{ {10}^{1.999.998 } }\% [/tex], maka pernyataan bahwa barang yg seharga Rp10.000,00 menjadi gratis jika menggunakan seluruh kupon adalah SALAH.
