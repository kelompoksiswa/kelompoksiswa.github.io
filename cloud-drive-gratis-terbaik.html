3. kesimpulan dari Google drive dan cloud computing​


Jawaban:

Google Drive adalah salah satu layanan penyimpanan berbasis cloud yang disediakan oleh Google. Dalam penggunaannya, pengguna dapat menyimpan berbagai jenis file seperti dokumen, gambar, video, dan lain sebagainya. Selain itu, Google Drive juga memungkinkan pengguna untuk berkolaborasi dengan pengguna lainnya dalam membuat dan mengedit dokumen secara bersama-sama.

Sementara itu, cloud computing adalah sebuah konsep teknologi yang memungkinkan akses ke sumber daya komputasi, seperti penyimpanan, aplikasi, dan infrastruktur jaringan melalui internet, sehingga pengguna tidak perlu memiliki perangkat keras dan infrastruktur yang mahal. Cloud computing memungkinkan pengguna untuk memperoleh sumber daya IT sesuai dengan kebutuhan mereka dan membayar hanya untuk sumber daya yang mereka gunakan.

Kesimpulan dari Google Drive dan cloud computing adalah bahwa keduanya merupakan teknologi yang berbasis cloud. Google Drive adalah salah satu contoh layanan penyimpanan cloud yang dapat diakses melalui internet, sedangkan cloud computing merupakan sebuah konsep teknologi yang memungkinkan akses ke berbagai sumber daya IT melalui internet. Keduanya memberikan keuntungan bagi pengguna, seperti kemudahan akses dan kolaborasi, penghematan biaya, dan fleksibilitas dalam penggunaan sumber daya IT.

8. private cloud public cloud hybrid cloud merupakan​


Jawaban:

Public cloud adalah cloud computing yang dikirimkan melalui internet dan dibagikan ke seluruh organisasi atau perusahaan. Private cloud adalah cloud computing yang didedikasikan hanya untuk organisasi atau perusahaan Anda. Hybrid cloud merupakan gabungan antara public cloud maupun private cloud.
