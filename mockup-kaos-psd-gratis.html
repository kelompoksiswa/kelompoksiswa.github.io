5. pengertian format gambar PSD,EPS,JPEG,GIF,PNG,TIFF,BMP


JPEG (Joint Photographic Expert Group) adalah standar kompresi file yang dikembangkan oleh Group Joint Photographic Experts; menggunakan kombinasi DCT dan pengkodean Huffman untuk mengkompresikan suatu file citra. JPEG adalah suatu algoritma kompresi yang bersifat lossy, (yang berarti kualitas citranya agak kurang bagus).

Graphics Interchange Format(GIF) adalah sebuah format yang sering digunakan dalam dunia web maupun dalam dunia citra digital. Format ini sering digunakan karena ukurannya yang relati kecil dan juga banyaknya software editor gambar yang telah mendukung citra ini. GIF berukuran kecil karena membatasi jumlah warnanya sebanyak 256 warna sehingga dapat menghemat ukuran berkas. 

BMP merupakan file gambar yang tidak terkompresi. Dan karena itu ukuran file BMP biasanya besar. File berukuran besar tidak tepat untuk transfer di internet karena akan memenuhi bandwidth dan menjadi lambat.

The TIFF (Tagged Image File Format) adalah format gambar yang fleksibel biasanya menyimpan 16-bit per warna - merah, hijau dan biru untuk total 48-bit - atau 8-bit per warna - merah, hijau dan biru untuk total 24-bit - dan menggunakan nama file atau perpanjangan TIFF TIF. TIFF yang kedua adalah fleksibilitas fitur, dan kutukan, dengan tidak ada satu pembaca semua mampu menangani berbagai jenis file TIFF. TIFF dapat lossy atau lossless. 

(Portable Network Graphics) adalah salah satu format penyimpanan citra yang menggunakan metode pemadatan yang tidak menghilangkan bagian dari citra tersebut (Inggris lossless compression).

8. Mungkin ada yang bisa menjelaskan sesuai dengan pemahaman kalian tujuan dibuat studi model/mockup dalam membuat produk?​


Penjelasan:

Mockup atau model adalah contoh atau prototipe dari sebuah produk yang belum dirilis ke publik atau konsumen. Dengan kata lain, mockup itu juga merupakan acuan dalam mewujudkan ide-ide inventor, contohnya seperti pada perancangan desain website. Sebelum merancang sesuatu, kita memerlukan contoh atau model sebagai imajinasi dari produk yang ingin dibentuk. Dari contoh tersebut, barulah perancang atau inventor tersebut bisa mempresentasikan kepada teamnya apakah model tersebut bisa dipergunakan atau tidak. Ini dikarenakan jikalau tidak ada model, maka itu akan merugikan pihak perancang yang sudah memproduksi produk tersebut ke publik tanpa uji coba.

Untuk itu, banyak perusahaan besar menggunakan mockup atau prototype ini sebagai acuan mereka sebelum produk dirilis (launching). Tanpa penggunaan prototype, perusahaan dapat rugi jutaan rupiah jika perilisan produk akhir tidak sesuai dengan tujuan awal pembuatan produk tersebut.

15. Tuliskan lima contoh dari psd png dan tiff singkat tepat padat


Jawaban:

Macam - macam jenis format gambar antara lain :

Photoshop Document ( PSD )

Encapsuled Postscript ( EPS )

Joint Photographic (Expert) Group ( JPG/JPEG )

Portable Network Graphic ( PNG )

Pict ( PIC )

Bitmap Image ( BMP )

Pembahasan

Photoshop Document ( PSD )

Format gambar jenis ini berupa dokumen grafis yang dapat dibuka dengan Adobe Photoshop. Di dalamnya tersimpan layer dan channel gambar untuk dilakukan pengubahan.

Encapsuled Postscript ( EPS )

Format gambar jenis ini juga berupa dokumen grafis namun lebih populer karena dapat dikenali oleh sebagian besar software untuk mencetak gambar. Umunya fprmat ini digunakan oleh para desainer grafis sebagai format penyimpanan final.

Joint Photographic (Expert) Group ( JPG/JPEG )

Format gambar jenis ini adalah format yang paling banyak digunakan karena kemampuannya untuk mengatur kualitas sesuai dengan yang diingikan. Mode warna yang disedikan juga dapat dalam RGB, CMYK dan grayscale.

Portable Network Graphic ( PNG )

Format gambar jenis ini merupakan bentuk lain dari format GIF. Pengguna dapat menyesuaikan sendiri kualitas gambar dan format ini dapat menyimpan bit depth sebesar 24 bit. Keunggulan utama format ini adalah background yang disediakan transparan.

Pict ( PIC )

Format gambar jenis ini mirip dengan PSD maupun EPS. Hanya saja format gambar ini merupakan format standar yang berguna untuk program gradis yang ada dalam sistem operasi Mac.

Bitmap Image ( BMP )

Format gambar ini merupakan format yang standar bagi pengguna Microsoft Windows karena sifatnya yang dapat diibaca semua program grafis. Mirip dengan PIC, format ini hanya berfungsi dengan baik pada sistem operasi masing - masing.
