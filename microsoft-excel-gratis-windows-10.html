13. bagaimana cara menjalankan Microsoft Excel yang terdapat pada Windows Explorer


Jika ingin membuka Microsoft Excel dengan cara pertama, maka pada desktop harus sudah ada shortcut-nya terlebih dahulu. Seandainya shortcut belum terpasang di desktop, shortcut dapat dimunculkan dengan langkah berikut.

1. Klik kanan pada desktop.
2. Pilih menu New, kemudian klik Microsoft Excel worksheet.
3. Klik (2x) Microsoft Excel untuk membukanya
- Buka Windows Explorer 
- Kemudian cari direktori/folder yang memuat file lembar kerja Excel (biasanya berakhiran .xls atau .xlsx) 
- Lalu cari file lembar kerja Excel tersebut 
- Kemudian klik 2x file lembar kerja Excel tersebut, dan nanti akan muncul jendela Excel 
Atau bisa juga dengan masuk ke folder tempat dimana program Excel berada:
- Buka Windows Explorer 
- Masuk ke Local Disk C, lalu ke folder Program Files, lalu ke folder Microsoft Office, lalu cari program bernama excel.exe, kemudian klik 2x file excel.exe dan tunggu hingga jendela Excel muncul
