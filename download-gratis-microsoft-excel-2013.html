24. Pengertian ekstensi XLA pada Microsoft Excel 2013


Pengertian ekstensi XLA pada Microsoft Excel 2013, pengertian dari ekstensi file XLA pada excel 2013 adalah ekstensi add in versi excel 97-2003. Add in sendiri merupakan salah satu komponen pada microsoft excel yang tidak otomatis terpasang ketika kita menggunakan excel. Untuk menggunakan add ini kita bisa menambahkannya terlebih dahulu. Dengan menggunakan add in, suatu perintah pada microsoft excel dapat berjalan secara otomatis ketika syaratnya terpenuhi.

Selain ekstensi file .xla, pada microsoft excel juga memiliki ektensi file lainnya, salah satunya adalah ekstensi xlsx untuk format file excel versi 2007 ke atas. Sedangkan untuk microsoft excel 2007 ke bawah masih menggunakan ektensi file .xls. Untuk ekstensi file dokumen yang lain, masih ada docx untuk microsoft word dan pptx untuk dokumen power point.
