Jawaban:

1. Prediksi Penjualan: Penggunaan WEKA untuk memprediksi penjualan produk berdasarkan faktor-faktor seperti harga, lokasi, promosi dan lainnya. Pengguna dapat menggunakan algoritma WEKA seperti algoritma klasifikasi, regresi atau clustering untuk menganalisis data dan membuat prediksi.

2. Analisis Sentimen: Penggunaan WEKA untuk menganalisis sentimen di berbagai platform media sosial seperti Twitter, Facebook dan Instagram. Pengguna dapat menggunakan algoritma WEKA seperti Naive Bayes atau J48 untuk mengklasifikasikan komentar sebagai positif atau negatif.

3. Prediksi Penggunaan: Penggunaan WEKA untuk memprediksi penggunaan layanan atau produk berdasarkan data historis. Pengguna dapat menggunakan algoritma WEKA seperti regresi logistik atau algoritma klasifikasi untuk membuat prediksi yang akurat.
