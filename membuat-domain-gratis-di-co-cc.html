12. domain pada internet-CO-ID-COM-EDU-ORG-HTML-AU-ONIONJawab Plis​


Domain pada internet

".co" domain untuk beberapa negara. ( Jika di Indonesia menggunakan .co.id, namun negara kolombia menggunakan langsung .co )

".id" domain untuk negara Indonesia

".com" domain internasional

".edu" domain untuk pendidikan

".org" domain untuk organisasi

".html" domain untuk tipe ekstensi yang berbentuk html, domain ini juga bisa berupa file dan bisa dijalankan offline

".au" domain untuk negara australia

".onion" domain untuk pengguna aplikasi khusus ( Dalam hal ini butuh aplikasi yang bernama TOR ) terkadang disebut domain untuk dark web

Pembahasan

Yang kita liat diatas ialah top level domain, domain diatas berarti domain yang tidak bisa ditambahkan lagi. Sebagai contoh domain pada web brainly.co.id ialah ".co.id", ".co" adalah second level domain sedangkan ".id" adalah top level domain.

Diatas juga merupakan domain yang bisa digunakan sebagai website dan gmail ( Kecuali .html dan .onion ). Fungsi domain cukup penting sebagai identitas suatu organisasi ataupun website.

Nama domain merupakan nama unik buat mengidentifikasi alamat IP yang berupa kombinasi nomor  yg rumit. Dengan nama domain ini, user akan lebih gampang buat mengingat alamat web Anda. Domain name jua tak jarang diklaim menjadi alamat website. Domain merupakan nama unik yang diberikan buat mengidentifikasi alamat (IP address) server personal komputer  misalnya web server atau email server pada internet.
