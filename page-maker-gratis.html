30. aplikasi yang digunakan sebagai media hiburan untuk bermain game adalah paradox Adobe ReaderPage maker game​


Aplikasi yang digunakan sebagai media hiburan untuk bermain game adalah game. Game adalah aplikasi atau program komputer yang dibuat untuk menghibur dan memberikan pengalaman bermain kepada penggunanya. Game biasanya terdiri dari sejumlah aturan dan tujuan yang harus dipenuhi oleh pemain, serta menyediakan mekanisme interaksi yang memungkinkan pemain untuk berkomunikasi atau bersaing dengan pemain lain. Game dapat dimainkan di berbagai perangkat, seperti komputer, laptop, atau smartphone, dan dapat diunduh atau dibeli melalui toko aplikasi atau situs web game.

Sementara itu, aplikasi paradox adalah program database yang digunakan untuk membuat, mengelola, dan memanipulasi data. Adobe Reader adalah aplikasi yang digunakan untuk membuka dan membaca file PDF. Page maker adalah aplikasi desain grafis yang digunakan untuk membuat dan mengedit dokumen atau publikasi seperti buku, majalah, atau brosur.
