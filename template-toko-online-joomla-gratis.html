26. jabarkan tentang aplikasi CMS Joomla​


Jawaban:

Jommla yang awalnya dibuat oleh perusahaan Miro yang berasal dari Australia yang dalam pengembangannya berawal dari poject open source kemudian beralih ke mambo CMS seiring dengan perkembangannya mambo berubah menjadi Joomla yang berjalan sampai sekarang

Joomla salah satu aplikasi CMS atau yang biasa di sebut “Content Management System” yang open source berarti tidak di kenakan biaya sepeserpun alias gratis untuk mendapatkan fasilitas joomla.

Joomla terdiri dari 3 elemen dasar, yaitu “server web” skrip “PHP” dan basisdata “MySQL“. Server web diasumsikan terhubung dengan Internet/Intranet yang berfungsi sebagai penyedia layanan situs. Skrip PHP terdiri dari kode program dalam bahasa PHP dan basisdata merupakan tempat penyimpanan konten. Joomla menggunakan Apache sebagai server web dan MySQL untuk basisdatanya.

28. Bagaimana cara memilih template template Microsoft publisher?


Pelajaran TI
=================
Setiap kali Anda membuka Publisher 2013 atau memilih File ➪ New [CTRL + N], Publisher akan menampilkan pilihan galeri New Template di Backstage View sehingga Anda bisa mencari dan memilih sebuah template untuk digunakan membuat file publikasi baru. Ketika komputer Anda terhubung dengan internet dan Sign In ke Publisher, Anda bisa klik Featured diatas thumbnail template dan Scroll/menggulung kebawah untuk melihat template lain yang tersedia untuk di-download. Klik thumbnail template untuk melihat jendela preview dan deskripsinya, lalu klik Create untuk mendownload template tersebut dan membuat publikasi baru menggunakan template tersebut.

29. bagaimana cara membuat poling di website joomla?


joomla versi berapa ?

untuk joomla versi 3.2 dan 2.5 kamu bisa menginstall module polling dari menu :

Extensions - Extension Manager

untuk modulenya kamu bisa dapatkan di website resmi joomla.org

selamat mencoba, jika ada kendala silahkan posting disini biar dibantu
Download dulu component AcepollsSetetlah di Download, Login ke AdminKlik menu Extension – Extension Manager lalu upload dan instal Component AcePolls yang sudah anda Download tadi.Jika instal berhasil, pilih menu Component dan klik AcePolls Lalu anda tinggal buat Polling anda.Selesai, sekarang polling anda sudah jadi, sekarang tinggal  bagaimana cara agar polling yang telah anda buat tadi muncul pada website.Balik lagi ke menu Extension – Module  ManagerKlik new untuk membuat module baru, lalu pilih AcePolls pada type module.Nah sekarang anda tinggal berikan judul seperti Polling Pengunjung dan atur dimana anda akan letakan Module Polling anda tadi pada website
