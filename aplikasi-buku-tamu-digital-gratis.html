6. Aplikasi yang digunakan untuk menyusun buku digital dengan format epub adalah ....


Sigil, Scribus, Mobipocket Creator, Calibre, Scriba, ePUBee Maker dan Microsoft Word.

Penjelasan detail :

Aplikasi yang Digunakan untuk Menyusun Buku Digital dengan format epub sebagai berikut :

1. Sigil

Sigil adalah aplikasi yang bisa digunakan untuk menyusun buku digital dengan format beragam. Saking bagusnya, aplikasi ini memiliki jumlah pengguna terbanyak di seluruh dunia. Software esensial yang satu ini juga mendukung fitur multiplatform, di mana kamu bisa menggunakan aplikasi ini di Windows, Linux, dan Mac.

2. Scribus

Rekomendasi aplikasi editor EPUB yang berbasis open source adalah Scribus. Aplikasi ini pun sudah bisa digunakan di berbagai OS komputer. Karena berbentuk open source, artinya kamu bisa menggunakannya secara gratis tanpa harus mengeluarkan uang sama sekali dan masih terhitung tindakan yang legal.

3. Mobipocket Creator

Program yang satu ini juga telah banyak digunakan di seluruh dunia untuk membuat EPUB. Melalui program ini, kamu bisa membuat buku digital yang kamu idam-idamkan dengan memanfaatkan berbagai macam fitur menarik yang ditawarkan aplikasi. Mobipocket Creator juga bisa meng-import berbagai macam dokumen seperti PDF, HTML, dan doc yang hasilnya bisa digunakan sebagai bahan untuk menulis.

4. Calibre

Aplikasi ini tergolong sangat komplit untuk urusan penyusunan buku digital. Di aplikasi ini kamu bisa membuat buku digital dalam bentuk EPUB, mengatur koleksi buku digital yang kamu miliki, dan tersedia pula dukungan bahasa yang terbilang sangat beragam. Calibre pun dapat membaca beragam for mat dokumen yang ada seperti PDF, DOC, TXT, HTML, dan file dokumen dalam bentuk lainnya.

5. Scriba

Salah satu aplikasi pengolah buku digital adalah Scriba. Selain bisa membuat file EPUB, kamu juga bisa membuat buku digital lain dalam bentuk PDF atau MOBI. Tampilan aplikasi pengolah kata ini terbilang sangat menarik. Kamu bisa menulis sambil melihat preview buku dalam bentuk 2 page preview.

6. ePUBee Maker

ePUBee Maker akan membantu kamu untuk membuat buku digital dalam format EPUB, Mobi, hingga PDF. Tak hanya itu, aplikasi ini juga mengklaim mampu menghasilkan kualitas yang sama seperti dokumen Word Jika ingin mengubah jenis format buku digital ke Word, aplikasi ini juga menawarkan fasilitas tersebut. Kamu bisa mengimport file EPUB, MOBI, AZW, AZW3, HTML, hingga TXT ke format Word.

7. Microsoft Word

Program ini awalnya hanya tersedia untuk sistem operasi Microsoft Windows saja. Tapi sekarang software ini juga sudah tersedia untuk sistem operasi Apple MacOS. Sejak versi Office 2013, MS. Word sudah bisa digunakan untuk mengedit dan menyimpan file dalam bentuk PDF. Selain itu, file .DOC atau .DOCX yang sudah jadi pun bisa dikonversi menjadi PDF juga.

Jadi, aplikasi yang digunakan untuk menyusun buku digital dengan format epub adalah Sigil, Scribus, Mobipocket Creator, Calibre, Scriba, ePUBee Maker dan Microsoft Word.

22. aplikasi yang di gunakan untuk menyusun buku digital


Saya akan mencoba menjawab dengan dua jawaban:

Kelas: VII
Mata pelajaran: Teknologi Informasi
Materi: Pengolah Kata/Word Processor

Jawaban pendek:

Aplikasi yang digunakan untuk menyusun buku digital adalah aplikasi pengolah kata seperti Microsoft Word, iWork Pages, OpenOffice Writer, LibreOffice Word dan Adobe Acrobat.

Jawaban panjang:

Buku digital atau e-book dapat dibyat dengan berbagai applikasi pengolah kata atau word processor. Format file yang digunakan sebagai penyimpanan buku digital adalah .pdf (portable document format), .epub (electronic publishing) dan .mobi (mobipocket e-book).

Aplikasi yang dapat digunakan untuk menyusun buku digital dan menyimpan dalam format diatas adalah:

1. Microsoft Word:

Program ini adalah pengolah kata yang paling banyak dipakai. Word adalah bagian dari keluarga perangkat lunak kantor Microsoft Office. Word dapat mengedit dan menyimpan dalam bentuk PDF sejak versi Office 2013. Program ini awalnya hanya tersedia untuk sistem operasi Microsoft Windows, namun sekarang juga tersedia untuk sistem operasi Apple Mac OS.

2. iWork Pages:

Pengolah kata ini adalah program bagian dari iWork, keluarga program perkantoran yang dirilis oleh Apple sebagai pesaing Microsoft Office. Seperti Word, program ini juga mampu menyimpan file dalam format pdf dan epub. Program ini tersedia untuk sisten operasi iOS (smartphone) dan Mac OS (desktop).

3. OpenOffice Writer dan LibreOffice Writer:

OpenOffice dan LibreOffice adalah dua program yang berhububgan erta. LibreOddice merupakan percabangan (fork) dari OpenOffice yang dibuat pada tahun 2010 ketika OpenOffice dibeli oleh perusahaan Apache. Keduanya merupakan upaya komunitas open souce untuk membuat alternatif gratis dan bebas dari Microsoft Word. Keduanya mampu menyimpan buku digital dalam format PDF. Kedua program ini tersedia untuk sistem operasi Windows, Linux dan Mac OS.

3. Adobe Acrobat

Adobe Acrobat adalah program yang merupakan program pertama untuk membuat dan mengedit file buku digital dalam bentuk PDF. Program ini berbayar, versi gratisnya adalah Adobe Reader yang hanya mampu membaca PDF saja tanpa bisa mengeditnya.

26. Untuk membuat daftar isi pada buku digital dengan menggunakan aplikasi sigil adalah ...


Sigil adalah program editor buku digital yang dikembangkan oleh Strahinja Markovic pada tahun 2009 dan dikelola oleh John Schember sejak tahun 2011. Sigil mendukung penggunaan WYSIWYG (What You See Is What You Get) dan mengetik kode berbasis file EPUB, serta impor HTML dan file teks biasa.


Untuk membuat daftar isi pada buku digital menggunakan aplikasi sigil, kita perlu merapikan terlebih dahulu isi materi kita dengan menambahkan tag H1, H2, H3, OL, dan lainnya sesuai dengan keinginan. Setelah semua rapi, klik Alat - Daftar Isi - Menghasilkan Daftar Isi. Dapat juga menggunakan kombinasi tombol Ctrl + T. Jika muncul kotak banner, klik Ok untuk melanjutkan pembuatan daftar isi.
