3. ka kenapa google chrome aku ngga ada jadinya dregol penelusuran bukan google penelusuran apa itu gara gara aku sering download aplikasi aplikasi gitu?


oh aku tau kok dregol :)
itu biasanya gara2 kamu waktu install sesuatu diminta install itu dan kamu oke

cara benerinnya agar ribet, tapi ikuti semua langkah2nya ya

A
1. kamu buka start > ketik "program" di search terus pilih program and features
2. kamu cari program yang ada "dregol" di namanya, terus kamu uninstall semua

B
1. kamu klik ikon 4 garis di kanan atas
2. pilih more tools
3. pilih extensions
4. hapus "dregol" (ikon sampah)
5. restart chrome

C
1. kamu klik lagi ikon 4 garis di kanan atas
2. pilih setting
3. pilih on start up, terus ke set pages
4. kalau ada "dregol", dipilih, terus remove (x)
5. setelah itu kamu pilih "manage search engine"
6. dan pilih google.co.id (atau situs lain yang kamu mau)
