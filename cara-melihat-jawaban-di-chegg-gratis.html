Penjelasan dengan langkah-langkah:

4x² + 20x - 24 = 0 (sama2 dibagi 4)

x² + 5x - 6 = 0

(x + 6) (x - 1) = 0

x + 6 = 0 atau x - 1 = 0

x1= -6 atau x2 = 1

misal:

m = x1 = -6

n = x2 = 1

maka,

a) m² + n²

= (-6)² + 1²

= 36 +1

= 37

b) 1/m + 1/n

= 1/(-6) + 1/1

= - 1/6 + 6/6

= 5/6

c) 1/m² + 1/n²

= 1/(-6)² + 1/1²

= 1/36 + 1/1

= 1/36 + 36/36

= 37/36

= 1 1/36

d) m² + n² - (1/m + 1/n)²

= 36 + 1 - (5/6)²

= 37 - 25/36

= 1332/36 - 25/36

= 1307/36
