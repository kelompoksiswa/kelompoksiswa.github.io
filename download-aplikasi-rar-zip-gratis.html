30. Di ketahui suatu server (debian), memiliki suatu file berkestensi .zip dan .rar, agar file tersebut bisa digunakan dan di pindahkan ke folder root /var/www/html langkah apa yang harus di lakukan oleh administrator server tersebut?​


Langkah yang harus di lakukan oleh administrator server tersebut adalah dengan cara mengobah alamat menjadi /home/(nama_user)/public_html/>. Lalu setelah itu lakukan sudo restart.

Pembahasan

Server adalah sistem komputer yang memiliki layanan khusus berupa penyimpanan data. Peran server sangat penting untuk mengirim atau menerima data dan informasi yang tersedia.Pada umumnya tugas utama komputer server adalah melayani dan bertanggung jawab penuh atas permintaan data dari komputer client. Ini juga berperan sebagai pengatur hak akses jaringan yang dapat digunakan komputer klien. Server adalah sistem komputer yang menyediakan sumber daya untuk menyimpan data. Biasanya, penyimpanan data di server digunakan untuk menyimpan dokumen dan data yang digunakan untuk melakukan berbagai layanan. Misalnya melihat website, mengirim email dan lain-lain.
