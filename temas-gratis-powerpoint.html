Jawaban:

Perbedaan mendasarnya dapat diketahui saat masing-masing file dibuka:

- saat type file powerpoint presentation dibuka, maka tampilannya standar (pilihan slide di panel kiri & pilihan isi dari slide di panel kanan)

- saat type file powerpoint show dibuka, maka tampilannya adalah slide show (yg secara otomatis) dimulai dari slide awal sampai slide akhir
