19. Sebuah mainan mobil truck mempunyai bagasi berbentuk balok mempunyai luas permukaan696 cm? Jika panjang bagasi 18 cm dan lebar bagasi 6 cm, maka tinggi bagasi tersebut adalahjawab:cm​


Jawaban:

Lp balok = 2 x ( ( p x l ) + ( p x t ) + ( l x t ) )

696 = 2 x ( ( 18 x 6 ) + ( 18 x t ) + ( 6 x t ) )

696 = 2 x ( 108 + 18t + 6t)

696 = 216 + 36t + 12t

696 - 216 = 36t + 12t

480 = 48t

t = 480/48

t = 10 cm
