7. Uang Adinda Rp40.000,00 lebih banyak dari uang Binary ditambah dua kali uang Cindy. Jumlah uang Adinda, Binary, dan Cindy Rp200.000,00, selisih uang Binary dan Cindy Rp10.000,00. Jumlah uang Adinda dan Binary adalah ….


Misal :
uang adinda = a
uang binary = b
uang cindy = c
jumlah uang Adinda dan Binary = ?

a = (40.000 + b) + (2 × c)
a + b + c = 200.000
b - c = 10.000 ........ b = 10.000 + c
a + b = ?

maka 
a = (40.000 + b) + (2 × c)
a = 40.000 +(10.000 + c) + 2c
a = 50.000 + c + 2c
a = 50.000 + 3c

substitusi
a + b + c = 100.000
(50.000 + 3c) +(10.000 + c) + c = 200.000
60.000 + 5c = 200.000
5c = 200.000 - 60.000
5c = 140.000
c = 140.000/5
c = 28.000

b = 10.000 + c
b = 10.000 + 28.000
b = 38.000

a = 50.000 + 3c
a = 50.000 + (3×28.000)
a = 50.000 + 84.000
a = 134.000

sehingga a + b = 134.000 + 38.000
= 172.000

jumlah uang adinda dan binary 172.000
