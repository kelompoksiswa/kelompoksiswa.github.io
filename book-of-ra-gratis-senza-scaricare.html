22. 3. The page of mathematics book starts from 1. If there are 852 digits used in the book, then the number of the last page of the book is ....?​


Jawab:

﻿320 pages

Unit digits:

Page 1-9 = 9×1 = 9 digits

Tens digits:

Page 10-99 = 90×2 = 180 digits

The rest digits:

= 852 - (9+180)

= 663

The hundred digits:

= 663 ÷ 3

= 221 pages

The total pages:

= 99 + 221

= 320
