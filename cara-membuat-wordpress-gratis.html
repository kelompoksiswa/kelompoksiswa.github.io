8. Berikut merupakan tatacara membuat landing page di web wordpress, kecuali ? A. Beli domain dan hosting B. Install wordpress C. Install plugin wordpress D. Install framewoork


Tatacara membuat landing page di web wordpress, kecuali D. Install framework. Landing page ialah salah  satu jenis laman website yang didesain buat tujuan khusus, yaitu menaikkan angka penjualan serta leads atau prospek calon pembeli.

Pembahasan

Cara membuat Landing page pada WordPress

Log in atau Sign up WordPress. Install Plugin Elementor page Builder. Bikin Desain Landing halaman. memasukkan Elemen pada Landing page. Landing halaman Siap dipergunakan.

Landing page ialah salah  satu jenis laman website yang didesain buat tujuan khusus, yaitu menaikkan angka penjualan serta leads atau prospek calon pembeli. di sini, pengunjung yang mengklik iklan kita akan melihat kontennya, lalu melakukan action pada dalamnya sinkron menggunakan call-to-action (CTA) iklan tadi.

Contoh action-nya bermacam-macam, mampu membeli produk kita, berlangganan newsletter harian, memutar video, mengunduh modul secara online, serta lain-lain. Singkatnya, landing halaman ini ialah keliru satu tool pemasaran yang mampu mengubah pengunjung sebagai pelanggan atau konsumen.

Landing halaman bisa dibuat buat mendatangkan traffic ke website atau bahkan channel YouTube kita lewat iklan atau link yang dipasang di paid marketing (ads), e-mail, media sosial, serta sejenisnya. Selain itu, kita mampu memakai landing halaman buat mengkonversi pengunjung tersebut menjadi konsumen atau pengguna produk.

Jangan lupakan juga pengaturan tombol call-to-action pada dalamnya, jadi pengunjung mampu mengetahui tombol yang tepat untuk melakukan konversi. Tombol CTA di landing page ini mampu bervariasi sinkron dengan tujuannya, bisa berupa tombol pembelian, mengisi formulir, memainkan video attachment, dan  sejenisnya. agar lebih mudah dikenali pengunjung, tombol CTA sebaiknya diletakkan pada lokasi yang tepat dan  menggunakan desain yang menonjol.

9. buatlah contoh website menggunakan wordpress tentang biodata diri seseorang​


Jawaban:

Langkah 1. Beli Nama Domain

Langkah pertama menuju pembuatan situs pribadi Anda adalah membeli nama domain.

Karena situs ini adalah tentang Anda, pilihan yang jelas adalah menggunakan nama Anda sendiri.

Step 2. Pilih Hosting

Setelah Anda memiliki nama domain Anda, Anda perlu membeli hosting untuk situs web Anda. Anda bisa membeli hosting seharga $5 / month dan dalam kebanyakan kasus, Anda bahkan bisa membeli nama domain dan hosting dari penyedia yang sama. Pastikan untuk memilih penyedia hosting yang menawarkan dukungan pelanggan sehingga Anda bisa mendapatkan bantuan segera jika Anda membutuhkannya.

Langkah 3. Install WordPress

Kebanyakan penyedia hosting saat ini memiliki satu klik installer untuk WordPress. WordPress sendiri cukup mudah dipasang secara manual, setup satu klik mempercepat prosesnya dengan cukup. Apapun host yang Anda pilih, prosesnya relatif sama.

Softaculous wordpress installer

Installer Website

Setelah Anda membeli paket hosting, perusahaan hosting Anda akan mengirimkan email selamat datang yang berisi semua informasi penting tentang hosting Anda. Ini biasanya menyertakan tautan untuk masuk ke cPanel atau dasbor hosting Anda, serta nama pengguna dan kata sandi Anda untuk cPanel.

Langkah 4. Pilih Theme Wordpress Profesional

Situs pribadi Anda perlahan-lahan datang bersama. Anda memiliki domain, Anda telah membeli hosting, dan Anda juga menginstal WordPress. Langkah selanjutnya dalam daftar kami adalah memilih tema untuk situs web Anda.

Wordpress memiliki banyak tema, baik yang gratis dan berbayar, bagaimanapun, tidak semua tema diciptakan sama. Jika Anda serius membuat situs web resume profesional, satu-satunya pilihan yang masuk akal adalah tema WordPress premium.

Best WordPress Themes on ThemeForest

Tema WordPress yang terbaik di ThemeForest.

Dengan memilih tema premium, Anda dapat yakin situs web Anda mengikuti praktik WordPress terbaik dan Anda bahkan bisa mendapatkan dukungan dari pengembang tema jika Anda memerlukan bantuan dalam menyiapkannya.

Selain itu, tema resume WordPress juga memiliki fitur yang diperlukan untuk menampilkan pendidikan, keterampilan, pengalaman masa lalu, dan contoh proyek masa lalu Anda.

Terlebih lagi, Anda bahkan bisa memasukkan prestasi dan penghargaan, serta testimonial dari atasan masa lalu atau profesor perguruan tinggi Anda untuk menunjukkan keahlian Anda.

11. cara membuat menu dan memposting artikel di wordpress ???


Masuk ke halaman administrator (http://[nama-blog].wordpress.com/wp-login.php). [nama-blog] merupakan nama domain Anda, misalnya http://denipratama.wordpress.com/wp-login.php . Isikan data username/email dan passwordnya.
Selanjutnya Anda akan berada pada halaman beranda wordpress dan arahkan pointer ke menu post


Selanjutnya Anda berada di halaman posting. Isikan Judul artikel pada kolom di bawah tulisan Add New Post. Selanjutnya Isikan konten-kontenya pada kotak yang telah disediakan. Anda bisa menggunakan huruf cetak tebal, miring dan lainnya sesuai menu yang disediakan. Untuk dapat membuka semua ikon, klik ikon yang terletak paling kanan yang berisikan titik-titik. Anda bisa mencoba sendiri semua tool-tool yang notabene sudah familiar karena sama halnya dengan menu yang ada di microsoft word. Anda juga dapat menambahkan gambar , lagu atau file lainnya dengan klik Add Media yang berada dibawah kolom judul artikel. Klik saja Select Files dan pilih gambar/file yang ingin diupload. Lalu klik media yang ingin diunggah (usahakan agar terlihat gambar centang. Lalu isikan data-data dari media (image/musik) yang telah diupload. Lalu klik Insert into post.
Setelah itu, isikan data yang berada di sebelah kanan berupa Categories dan Tags. Categories merupakan kategori web seperti film, musik, lainnya. Jika belum terdapat kategori tersebut, Anda dapat klik Add New Category dan isikan nama kategori dan klik Add New Category dibawahnya. Sedangkan Tags merupakan keyword artikel seperti misalnya jika category adalah denipratama portfolio, untuk keyword contohnya : website, denipratama. Isikan penghubung koma agar mesin secara otomatis menampilkan dua keyword. Lalu klik add. Misalnya ingin mengubah category, hanya lakukan centang kembali pada kategory dan pilih kategori lain atau add new category. Sedangkan mengubah tag yaitu dengan mengklik tanda ‘X’ pada tag yang akan dihapus dan lakukan add tag baru. Lalu isikan set feature image dibawah menu tag (set feature image bisa diisikan ataupun tidak tetapi alangkah baiknya diisi agar terlihat lebih bagus.


Selanjutnya klik publish. Untuk mengubah postingan. Caranya yaitu dengan klik All Post pada menu Post disebelah kiri lalu cari tulisan edit dibawah judul artikel yang akan di edit.cara posting:
1. Masuk ke wordpress.com
Masukkan username anda
Masukkan password anda
Klik tombol Login
2. Pada page my blog, klik Dashboard
3. Akan tampil menu-menu blog (pada bagian samping kiri). Untuk memulai mengisi klik pada menu post, lalu klik Add new.
4. Isikan judul artikel/tulisan pada kolom title,
5. Pada kolom artikel isi dengan tulisan yang akan kita posting, kita dapat melakukan pengeditan seperti pada editor biasa.
6. Untuk menambahkan gambar atau foto dari Komputer anda, klik di menuUpload/Insert, Add Image>> Lalu klik Select File>> Lalu pilih gambar yang akan di upload dari komputer anda>> Kemudian klik Insert to Post.
7. Jika Pengeditan telah selesai dan materi yang kita isikan sudah kita anggap lengkap, maka langkah terakhir adalah mempublikasikan artikel tersebut di internet. Caranya adalah dengan mengklik tombol Publish atau Tampilkan.
8. Setelah terpublikasi, maka akan tertampil form untuk mengisi artikel/tulisan yang lain. Jika kita tidak ingin mengisikan tulisan lagi dan ingin melihat hasil tulisan kita apakah telah terisi dengan benar, maka kita tinggal mengklik menu View Post.
9. Pada web blog kita, akan tampil artikel/tulisan yang baru saja kita buat.
