7. Sebuah komputer diberi program antivirus A, B, dan C. Program antivirus A dijalankan setiap 14 hari sekali. Program antivirus B dijalankan setial 15 hari sekali. Program antivirus C dijalankan setiap 21 hari sekali. Setiap berapa hari sekali ketiga program antivirus tersebut dijalankan bersama?


14 = 2 x 7
15 = 3 x 5
21 = 3 x 7

KPK = 2 x 3 x 5 x 7
        = 210 hari

Jadi ketiga antivirus tersebut akan dijalankan bersama setiap 210 hari sekali atau 7 bulan sekalipertanyaan jenis ini adalah dengan mencari KPKnya
A = 14 = 2 × 7
B = 15 = 3 × 5
C = 21 = 3 × 7

KPK = 2 × 3 × 5 × 7 = 210 hari sekali ketiga antivirus akan dijalankan bersama.

9. Apakah antivirus menjamin komputer aman dari virus? Dan bagaimana cara kerja antivirus?


Signature.
Ketika ada file yang discan, antivirus akan membandingkan isi file dengan malware signature yang ada di signature database. Jika ada file yang cocok dengan salah satu signature, maka antivirus akan mengetahui tipe malware tersebut dan prosedur yang harus dilakukan untuk menghindari infeksi.

Heuristic.
Heuristic memungkinkan software antivirus untuk mengidentifikasi malware jenis baru, bahkan tanpa adanya file virus definition. Sebagai contoh, jika antivirus memberitahu ada aplikasi yang berjalan di sistem dan tengah mencoba untuk membuka setiap file EXE di komputer kamu, menginfeksi dengan menulis salinan aplikasi asli ke dalamnya, antivirus akan mendeteksi aplikasi ini sebagai file baru, yang diketahui sebagai virus tidak dikenal.
