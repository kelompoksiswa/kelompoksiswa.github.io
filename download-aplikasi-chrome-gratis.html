A
1. kamu buka start > ketik "program" di search terus pilih program and features
2. kamu cari program yang ada "dregol" di namanya, terus kamu uninstall semua

B
1. kamu klik ikon 4 garis di kanan atas

3. pilih extensions

5. restart chrome

C
1. kamu klik lagi ikon 4 garis di kanan atas
2. pilih setting
3. pilih on start up, terus ke set pages
4. kalau ada "dregol", dipilih, terus remove (x)
