5. Kode ! (tanda seru) pada bahasa pemrograman c menunjukan operator logika


Terdapat beberapa jenis dari operator dalam pemrograman bahasa C, salah satunya adalah operator logika. Di dalam operator logika, kode ! (tanda seru) memiliki arti negasi/kebalikan.

Pembahasan

Di dalam bahasa pemrograman C, ada yang namanya operand dan operator. Nilai asal yang digunakan dalam suatu proses operasi disebut sebagai operand. Lalu, suatu instruksi atau petunjuk yang diberikan guna memperoleh hasil dari suatu proses disebut dengan operator.

Operator logika merupakan salah satu jenis operator. Di dalam operator logika, kode ! (tanda seru) akan mengahasilkan false apabila operand true. Jadi, kode ! akan membalikkan sebuah logika, contohnya !true akan menjadi false, lalu apabila !false akan menjadi true.

case '+':

hasil = angka1 + angka2;

case '-':

hasil = angka1 - angka2;

break;

case '':

hasil = angka1 * angka2;

cout << angka1 << " " << sign << " " << angka2 << " = " << hasil;

case '/':

hasil = angka1 / angka2;

cout << angka1 << " " << sign << " " << angka2 << " = " << hasil;

break;

default:

cout << "Tolong masukkan operator yang valid!" << endl;
