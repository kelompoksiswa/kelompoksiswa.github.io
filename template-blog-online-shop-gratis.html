10. Sebutkan langkah memasang template di blog!​


Penjelasan:

Cara Memasang Template Blogger:

Template yang sudah sobat download tadi biasanya berformat .xml. Berikut adalah cara memasang template tersebut ke dalam Blogger (Blogspot):

1. Masuk ke Blogger.com, lalu pilih menu Template.

2. Dipojok kanan atas pilih Backup/Restore.

3. Klik Choose File lalu pilih file XML (template blog) yang sobat download tadi.

4. Jika sudah klik Upload, maka secara otomatis template baru akan terpakai.

Itulah cara mudah untuk mengganti template pada Blogger,.

24. cara mengubah template blog dari hasil browsing


tinggal buka templateny, lalu klik enable editing.1. Login ke Blogger. Ketik www.blogger.com di address bar browser Anda.

2. Klik menu Template / Tampilan di kiri bawah Dashborad Blog.

3. Klik menu Backup/Restore (Cadangkan/Pulihkan) di kanan atas Dashboard Blog.

4. Backup Template lama Anda (optional) dengan klik "Download Full Template".

5. Klik Choose File / Pilih File.

6. Anda akan dibawa menuju folder di komputer untuk menemukan file template blog. Klik file ekstensi XML yang sudah didownload/diekstrak, lalu klik "Open".

7. Klik Upload (Unggah).

28. Jelaskan kekurangan blog gratis​


Jawaban:

Beberapa kelemahan blog gratis:

1. Nama blog yang panjang.

2. Adanya iklan yang muncul saat blog diakses.

3. Pemilik blog gratis tak sepenuhnya memiliki kendali atas blognya.

Penjelasan:

1. Contohnya saja memakai blog gratis dari blogspot, maka alamat blog akan diakhiri dengan blogspot.com dengan nama awal blog. Tentunya nama yang panjang ini dapat menyulitkan pengunjung blog untuk mengingatnya.

2. Tampilan iklan yang muncul saat blog diakses dapat mengganggu pengunjung blog.

3. Pemilik blog wajib mengikuti aturan-aturan tertentu dari penyedia layanan blog gratis. Bila secara tak sengaja melanggarnya, maka blog buatannya dapat dihapus oleh penyedia layanan blog tersebut.
