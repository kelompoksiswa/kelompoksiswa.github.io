3. Apa bedanya situs dengan website?


Situs adalah kumpulan halaman-halaman yg digunakan untuk menampilkan informasi/gambar (gerak)/suara/gabungan dari semua itu, sedangkan website adl kumpulan halaman-halaman situs.Situs:
Situs dapat diartikan sebagai kumpulan halaman-halaman yang digunakan untuk menampilkan informasi, gambar gerak, suara, dan atau gabungan dari semuanya itu baik yang bersifat statis maupun dinamis yang membentuk satu rangkaian bangunan yang saling terkait dimana masing-masing dihubungkan dengan link-link.
Website:
Website adalah kumpulan dari halaman-halaman situs, yang biasanya terangkum dalam sebuah domain atau subdomain, yang tempatnya berada di dalam World Wide Web (WWW) di Internet. Sebuah halaman web adalah dokumen yang ditulis dalam format HTML (Hyper Text Markup Language), yang hampir selalu bisa diakses melalui HTTP, yaitu protokol yang menyampaikan informasi dari server website untuk ditampilkan kepada para pemakai melalui web browser. Semua publikasi dari website-website tersebut dapat membentuk sebuah jaringan informasi yang sangat besar.
Halaman-halaman dari website akan bisa diakses melalui sebuah URL yang biasa disebut Homepage. URL ini mengatur halaman-halaman situs untuk menjadi sebuah hirarki, meskipun, hyperlink-hyperlink yang ada di halaman tersebut mengatur para pembaca dan memberitahu mereka sususan keseluruhan dan bagaimana arus informasi ini berjalan.
Beberapa website membutuhkan subskripsi (data masukan) agar para user bisa mengakses sebagian atau keseluruhan isi website tersebut. Contohnya, ada beberapa situs-situs bisnis, situs-situs e-mail gratisan, yang membutuhkan subkripsi agar kita bisa mengakses situs tersebut.

19. jelaskan anatomi sebuah alamat website/situs


Sebuah situs web harus memiliki unsur-unsur yang akan mempengaruhi "user experience" saat mengunjungi situs web, seperti :

1. Tampilan
Unsur yang berperan penting dalam menangani tampilan web terletak pada kode yang menggunakan perpaduan bahasa pemograman CSS dan HTML.
Kedua bahasa ini harus saling berpadu karena jika hanya HTML saja, tampilannya akan jelek dan UE akan menurun. Tampilan merupakan hal terpenting dalam sebuah situs web karena merupakan apa yang dilihat pertama oleh pengunjung.

Dan biasanya untuk memperbagus tampilan, digunakan juga animasi yang dibuat oleh script;

2. Script
Script atau kode lebih ditugaskan pada JavaScript. Kode-kode JS yang dibuat (Tentunya dibuat oleh pemilik web) akan menerima setiap masukan dari pengguna dan menentukan apakah keluaran dari skrip tersebut. Kode ini diperlukan untuk membuat animasi web, karena hanya tampilan saja sepertinya kurang menarik. Kode ini juga dapat menampilkan sesuatu ke layar, tanpa pengguna harus me-refresh halamn web.

3. Resource
Resource merupakan hal terpenting dalam web meski tidak semuanya ditampilkan secara langsung. Beberapa ada yang ditampilkan di layar seperti gambar, video, atau audio. Yang lain seperti file zip, exe, apk, hanya akan ditampilkan jika pengunjung mengunduhnya.

4. Header
Yang terakhir ini adalah keterangan unik tentang sebuah halaman web. Header tidak ditampilkan dilayar. Namun ini merupakan hal penting dalam web. Biasanya pada web digunakan tag <header> untuk menjelaskan header.
Header berisi metadata tentang sebuah halaman web seperti judul, deskripsi, tanggal dibuat, charset, dll. Header juga berisi sumber-sumber atau resource eksternal dari sebuah halaman web, seperti skrip yang digunakan, file kode css, dan media yang ditampilkan.
