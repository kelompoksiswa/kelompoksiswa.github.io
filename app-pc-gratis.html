4. cara jalanin aplikasi yang 32 bit jadi 64 bit gimana? soalnya PC ku compatible 64 bit tapi app nya kebuka 32 bit


Jawaban:

1. klik pencarian pada windows, kemudian cari Windows Feature

2. klik pada turn windows feature On or Off

3. kemudian klik pada opsi internet Information Service (IIS), kemudian klik OK dan tunggu prosesnya hingga selesai.

4. Cari lagi pada pencarian internet Information Service (IIS)

5. Disini akan ditampilkan nama desktop sobat masing-maing, kemudian klik panah kecil hingga muncul opsi yang lainnya, kemudian pilih yg Application Pools, lalu kalik kanan pada DefaultAppPools dan pilih Advanced Setting,

6. Disinilah letak untuk mengaktifkan fitur 32 bit agar bisa menginstall aplikasi 64 bit.

Silahkan pilih True pada Enable 32-Bit Application

Sekarang silahkan Reststart komputer/laptop, setelah komputer/laptop direstart silahkan coba lagi untuk menginstall aplikasi 32 bit pada windows 64 bit
