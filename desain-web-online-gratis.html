1. apa perbedaan web gratis dengan web berbayar


Jawaban:

Perbedaan web gratis dan berbayar yang pertama ada pada server yang digunakan. Bagi pengguna web gratis, mereka biasanya tidak memiliki pengetahuan tentang letak server ataupun kualitas serta teknologi yang digunakan. Jika situs yang anda miliki lebih bersifat personal, hal ini tidak akan menjadi masalah namun jika web anda digunakan untuk tujuan bisnis, anda wajib mengetahui seluk beluk server yang ditawarkan. Penyedia jasa hosting berbayar akan memberikan anda informasi tentang hal ini sehingga anda bisa memilih sendiri server yang paling tepat untuk bisnis khususnya jika berkaitan dengan lokasi penempatan server.

3. Yang di maksud desain CS5 dalam mendesain web adalah


Desain CS5 merupakan aplikasi pengolah grafis dari Adobe Photoshop berbasis bitmap yang memiliki kekuatan pada teknik seleksi, manipulasi efek hingga membuat warna sendiri dalam menciptakan desain.

Pembahasan

Website merupakan sarana untuk membangun bisnis online yang berisi berbagai hal seperti informasi dan interaksi dengan orang lain.

Web Desain adalah menggambarkan bagaimana tampilan isi suatu website atau situs yang berpengaruh terhadap kenyamanan dan ketertarikan pengunjung web.

Adobe Photoshop adalah perangkat lunak editor buatan Adobe Systems yang digunakan oleh fotografer digital dan perushaan iklan khusus untuk pengeditan foto atau gambar dan pembuatan efek.

Salah satu program yang digunakan untuk membuat desain web adalah Adobe Photoshop CS (Creative Suite) 5 yang dirilis pada tahun 2010. Adobe Photoshop CS 5 merupakan versi photoshop yang kedua belas.

Program berbasis bitmap ini dapat membantu dan memberikan kemudahan bagi para desainer web dalam membuat berbagai macam desain web dengan menuangkan ide dan imajinasinya untuk menciptakan sebuah karya grafis secara lebih optimal. Kita bisa mengolah foto, membuat kreasi button dan membuat teks yang menarik.
