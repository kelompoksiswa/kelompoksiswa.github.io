8. Sebuah software gratis, dapat di download dan digunakan oleh pengguna. Akan tetapi penggunaan software tersebut ada batas waktunya, disebut


disebut shareware...

Pengertian Shareware adalah software gratis, dapat di download dan digunakan oleh pengguna. Akan tetapi penggunaan software tersebut ada batas waktunya, jika pengguna merasa softwarenya bagus, maka diharuskan membeli. Shareware sering dibatasi lamanya waktu pakai (misalnya trial 30 hari), atau jumlah software tersebut dijalankan (misalnya 30 kali), atau feature-feature tertentu yang tidak bisa diakses. Sesudah masa ujicobanya berakhir, software bisa saja terkunci atau bisa saja tetap berfungsi sebagaimana mestinya dan tetap terbatasi.

9. Perbedaan software ag yang gratis dan berbayar


Jawaban:

Perbedaan software yg gratis dan berbayar

- software gratis, yaitu yg berhak cipta yg gratis digunakan tanpa batasan waktu, berbeda dari shareware yang mewajibkan penggunanya membayar

software gratis (freeware) juga didefinisikan sebagai program apapun yang didistribusikan gratis, tanpa biaya tambahan.

Free Software lebih mengarah kepada bebas penggunaan tetapi tidak harus gratis. Pada kenyataannya, namanya adalah karena bebas untuk mencoba perangkat lunak sumber terbuka (Open Source) dan di sanalah letak inti dari kebebasan.

Jadi free software tidak mengarah kepada gratis pembelian tetapi penggunaan dan distribusi. Begitu keluar dari lisensi kita juga dapat menemukan berbagai cara untuk mendistribusikan perangkat lunak, termasuk freeware, shareware atau Adware.

- software berbayar, digunakan untuk tujuan komersil, jadi setiap pengguna yg ingin menggunakan / mendapatkan software tersebut dengan cara membeli / membayar pada pihak yang mendistribusikannya. Pengguna yang menggunakan software berbayar umumnya tidak diijinkan untuk menyebarluaskan software tersebut secara bebas tanpa ijin ada penerbitnya.


10. cara buat album foto


saya hanya akan memberitahu cara membuat album foto zaman modern...
bahan:
1.kertas karton ( setebal 1.3/3.6 cm )
2.gunting
3.kertas albino ( kertas setipis 0.5 cm )
4.foto
5.model bingkai
6.mesin perakit

cara membuat:
1.pertama potong kertas karton sebesar 11x5 persegi. siapkan kertas albino secukup mungkin misalnya 56/120 ( tergantung oleh jumlah foto )
2.lalu pasangkan model bingkai kepada mesin perakit dan sambungkan pada kertas albino
3.nyalakan mesin perakit ( Fix-enator ) lalu masukan kertas karton. album akan diproses 15 - 30 menit atau lebih ( tergantung jumlah foto )
4. setelah selesai pasangkan kertas albino ke setiap halaman untuk melekatkan photo afar tak mudah lepas..
5.terakhir pasangkan semua foto ke setiap halaman. dalam memasang foto dapat mencapai 30 - 50 menit ( tergantung jumlah foto )

19. Cara membuat album foto dengan menambahkan album melalui menu


Jawaban:

1. buka PowerPoint

2.Klik tab Insert yang ada di dalam Images group lalu pilih Photo Album

3.Pada list Photo Album, klik opsi New Photo Album. Nantinya jendela kotak Photo Album baru akan muncul di layar PC

4.Jika sudah, klik File/Disk pada kotak dialog tersebut, maka kotak dialog Insert New Picture akan muncul

5.Pilih file gambar yang kalian inginkan. Untuk memilih semua file gambar kalian, masukkan ke dalam folder. Klik pada satu file gambar, tahan tombol Shift dan tekan tombol panah bawah untuk memilih semua file

6.Selanjutnya klik Insert

7.Untuk menempatkan kotak teks di tata letak, klik New Text Box pada kotak dialog Photo Album

Langkah-langkah membuat album foto pada cerita tadi adalah

1)Siapkan kertas karton terlebih dahulu sebanyak 5-7 lembar.

2)Potong kertas karton yang muat untuk 2-3 lembar foto.

3)Beri lubang pada pinggir kertas karton yang telah dipotong.

4)Tempelkan foto dan beri hiasan yang berbeda.

5) Lalu gabungan kertas dengan benang tambal menjadi satu.
