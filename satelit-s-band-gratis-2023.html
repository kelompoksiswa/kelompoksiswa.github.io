25. sebuah satelit mengorbit bumi yang memiliki jari jari 6,4 . 10pangkat2. jika kelajuan satelit 6.400 m/s. maka tinggi satelit dari permukaan bumi adalah


Diketahui:
R bumi = 6.400 km
v satelit = 6,4 km/s
g bumi = 10 m/s² = 0,01 km/s²

Ditanya:
Ketinggian orbit satelit (h)

Penyelesaian:

⇔ v = R x √ [ g / (R + h) ]
⇔ g / (R + h) ] = [v / R]²
⇔ R + h = g x [R / v]²
⇔ h = { g x [R / v]² } - R
⇔ h = { 0,01 x [6.400 / 6,4]² } - 6.400
⇔ h = { 0,01 x 1.000.000 } - 6.400
⇔ h = 10.000 - 6.400
∴ h = 3.600 km di atas permukaan bumi

29. sebuah satelit memiliki kecepatan 10.000 m/s. setelah 50detik,satelit tersebut memiliki kecepatan 5000 m/s. percepatan yang dialami satelit adalah....pliss kasih cara ya guys makasih tolong bantuu dongg :)​


Percepatan yang dialami satelit adalah -100 m/s²

PEMBAHASAN

Diketahui:

kecepatan awal satelit = vo = 10.000 m/s

selang waktu = t = 50 sekon

kecepatan akhir satelit = vt = 5000 m/s

Ditanyakan:

percepatan satelit = a = ?

Penyelesaian:

Untuk mencari percepatan yang dialami oleh satelit , kita bisa memakai rumus GLBB berikut ini :

[tex]a = \frac{v_t - v_o}{t}[/tex]

[tex]a = \frac{5000 - 10000}{50}[/tex]

[tex]a = \frac{- 5000}{50}[/tex]

[tex]a = \boxed {- 100 ~ \texttt{m/s}^2}[/tex]

Nilai percepatan negatif artinya arah percepatan berlawanan arah dengan arah kecepatan sehingga satelit mengalami perlambatan.
