Desktop

Merupakan suatu tampilan awal pada saat Windows selesai di booting(tampilan home saat memasuki windows.

Wallpaper

Gambar yang menempel diatas desktop. Windows menyediakan berbagai macam wallpaper bawaan sehingga kita dapat mengubahnya dengan mudah dan cepat.

Start Menu

Tombol Start yang berisi berbagai macam program yang telah kita install, menu-menu lain yang dapat diakses oleh pengguna(user), seperti misalnya mendengarkan musik, menjalankan program editor teks dan lain-lain.

Icon

Gambar-gambar kecil yang menempel diatas desktop yang diikuti dengan identitasnya. Tujuan dari icon ini adalah untuk memudahkan dalam mengaktifkan suatu program aplikasi. Icon ini dapat ditambahi ataupun dikurangi sesuai dengan kebutuhan kita. Kalau terlalu banyak icon pada desktop akan mengurangi keindahan dari desktop itu sendiri.

Task Bar

Batang panjang yang terdapat paling bawah desktop. Kita dapat memindahkan taskbar ke atas atau samping kanan ataupun kiri desktop sesuai dengan keinginan kita sendiri. Pada taskbar ini terdapat program yang kita minimize-kan, icon-icon lainnya seperti icon pengaturan volume dan jam yang aktif.

Quick Launch

Bagian bawah sebelah kiri , sebelah kanan Start yang berisi icon-icon untuk shortcut program yang telah kita definisikan. Shortcut adalah cara cepat untuk menjalankan prgoram/data yang disimpan di suatu direktori tertentu. Shortcut berisi path menuju ke data/program yang dimaksud dan disimbolkan oleh icon tertentu.

Kali ini saya akan memberikan Tips n Trik untuk menampilkan Funngsi Hibernate di Windows 8 / 8.1,
Apa sih Hibernate?
hemm..ada yang tau hibernasi?
Nah itu dia jawabannya, Hibernate di windows digunakan untuk Menyimpan seluruh proses ke hardisk agar laptop dapat dimatikan tetapi kita dapat melanjutkan aktifitas yang kita lakukan sebelumnya pada saat kita menghidupkan komputer kembali,ini berbeda dengan sleep, biasanya kebanyakan orang menggunakan fitur ini bertujuan untuk meninggalkan komputer/laptop dalam waktu yang lama. biasanya proses penghidupan komputer setelah Hibernate disebut Resuming Booting.
Sebenarnya fungsi Hibernate ini sudah ada pada zaman windows XP,Vista dan 7, entah kenapa fitur ini disembunyikan. nanti kita akan bahas

pertama buka Power Option Pada Control Panel atau klik bagian bawah taskbar pada logo battre
kemudian setellah terbuka Power Options, Pilih "Choose what closing the lid does" Selanjutnya setelah terbuka System Setting kita checklist pada bagian Hibernate di menu Shutdown Setting,
jika menu shutdown setting tersebut masih buram dan tidak dapat di checklist, maka klik dulu pada bagian "Change setting are currently unvailable" barulah bagian hibernate bisa di checklist. selanjutnya klik button Save Changes pada bagian paling bawah.Jika step diatas dilakukan dengan benar, maka Muncullah menu hibernate pada saat kita akan melakukan prosedur Shut Down.
Jika Sudah Muncul pilihan Hibernate seperti Pada contoh gambar diatas, berarti kita sudah berhasil menampilkan Menu Plihan Hibernate Pada Windows 8 atau Windows 8.1 
Demikian sedikit Tutorial cara menampilkan Tombol Hibernate pada Windows 8 dan Windows 8.1, Jika ada pertanyaan, Sobat bisa tinggalkan pesan melalui kotak komentar dibawah ini. Terimakasih dan semoga bermanfaat.
