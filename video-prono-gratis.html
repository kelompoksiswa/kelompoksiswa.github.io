7. CARA MENGEDIT VIDEO BAGI PEMULA MENGGUNAKAN ADOBE PREMIER CC!! gratis 20 poin... DI TUNGGU YA


Sekarang project ini mungkin masih kosong, kita perlu memasukkan video untuk mulai mengedit. Cara insert video ke dalam project ini dapat dilakukan dengan cara klik File -> Import… pada menu bar diatas (atau bisa dilakukan dengan cara menekan CTRL+I pada keyboard). Silahkan pilih video dan audio yang akan di eksekusi. Proses insert ini memerlukan waktu loading sebelum item benar-benar berhasil di import kedalam project, waktu proses ini tergantung dari jumlah dan besarnya file yang di import.

Video dan audio yang dipilih nantinya akan tampil di dalam tab Project : Nama Project, selanjutnya silahkan tarik video dan audio yang ingin di eksekusi ke timeline (baris waktu pengeditan, disebelah bawah kanan).

11. object prono[tex]8 \sqrt{45 - 8 \sqrt{20 + \sqrt{245 - 3 \sqrt{125} } } } [/tex]​


Penjelasan dengan langkah-langkah:

[tex]8 \sqrt{45 - 8 \sqrt{20 + \sqrt{245 - 3 \sqrt{125} } } } \\ = 8 \sqrt{9.5 - 8 \sqrt{4.5 + \sqrt{49.5 - 3 \sqrt{25.5} } } } \\ = 8.3 \sqrt{5} - 8.2 \sqrt{5} + 7 \sqrt{5} - 3.5 \sqrt{5} \\ = 24 \sqrt{5} - 16 \sqrt{5} + 7 \sqrt{5} - 15 \sqrt{5} \\ = (24 - 16 + 7 - 15) \sqrt{5} \\ = \sqrt{5} [/tex]
