17. cara mendownload aplikasi line pada laptop bagaimana ya? mohon bantuannya​


LINE tersedia sebagai aplikasi desktop dan UWP (Universal Windows Platform).
Versi desktop hanyalah versi LINE yang bisa digunakan di Windows 7, Windows 8, Windows 8.1, atau Windows 10, juga macOS. Namun versi UWP jauh lebih ringan meski hanya dapat digunakan untuk Windows 10 dan Windows 10 Mobile.
Install line versi desktop..


Kemudian install aplikasi LINE seperti biasanya meng-install program atau software di komputer. Klik Next, Next, Finish.

Kemudian login dengan username atau email yang kamu punya. Kamu bisa login di dua perangkat sekaligus seperti di PC atau smartphone. Masukkan email dan password LINE.
Belum punya akun LINE? Daftar saja dengan mengklik Sign up.
Kamu juga bisa memilih Log me in automatically agar akun LINE kamu tetap login secara otomatis ketika aplikasi LINE di PC terbuka.
Atau, pilih juga Run LINE when Windows starts agar aplikasi LINE terbuka secara otomatis ketika kamu menyalakan PC atau laptop.
Klik Login untuk masuk ke LINE di PC.
Ketika login, kamu mendapat pesan dengan angka di dalamnya. Angka ini adalah kode verifikasi yang harus dimasukkan pada aplikasi LINE di smartphone Android atau iPhone atau Windows 10 Mobile kamu.


Masukkan angka tersebut pada aplikasi LINE di smartphone, dan klik Verify


Setelah itu, kamu akan melihat pesan ini, yang berarti kode verifikasi benar dan telah berhasil login ke LINE di PC/laptop.


Kembali ke PC/laptop, dan kamu akan langsung melihat LINE di sana dengan history chat, daftar teman, dan lain sebagainya yang sama seperti pada LINE di smartphone.

29. Tulisankan langkah mengedit teks dgn aplikasi photo shop​


Cara Menambahkan Teks di Photoshop

Disusun bersama Staf wikiHow

Adobe Photoshop umumnya digunakan untuk membuat ilustrasi dan menyunting foto. Anda juga dapat menambahkan teks di Photoshop atau menyesuaikan properti seperti jenis huruf, ukuran, dan warna teks. Kemudian, Anda dapat membuat gambar, iklan, atau judul. Namun, ingatlah bahwa tujuan penambahan teks di Photoshop adalah memperkaya gambar dengan pesan singkat, alih-alih mengetik paragraf atau membuat dokumen yang hanya berisi teks.

Metode 1 dari 2:

Menambahkan Teks Apa Saja

1

Pilih Type tool dengan ikon huruf "T" dari bilah alat peralatan. Klik ikon tersebut atau tekan "T" pada papan tik untuk menampilkan alat teks. Kini, Anda dapat mengeklik bagian gambar mana saja untuk mulai mengetik.

2

Atur opsi teks dengan menu di bagian atas layar. Setelah mengeklik opsi teks, Anda akan melihat berbagai opsi di bagian atas jendela Photoshop, yang memungkinkan Anda mengubah warna, jenis huruf, ukuran, dan posisi teks. Anda juga dapat menggunakan "Character" atau "Paragraph", yang mirip dengan kotak teks di program seperti Word. Anda dapat menemukan kotak tersebut dengan mengeklik "Window" pada bagian atas layar dan memilih "Character" dan "Paragraph".[1]

Font: memungkinkan Anda memilih jenis huruf lain, seperti Arial dan Times New Roman.

Font Size: memungkinkan Anda memperbesar atau memperkecil teks.

Font Alignment: memungkinkan Anda membuat teks rata kiri, tengah, atau kanan.

Font Color: memungkinkan Anda memilih warna lain untuk teks.

3

Klik bagian gambar yang ingin Anda bubuhi teks di Photoshop. Jika Anda mengeklik bagian tertentu pada gambar, Anda akan melihat kursor di posisi huruf pertama. Anda dapat mulai mengetik, dan Photoshop akan menambahkan teks pada titik yang Anda pilih.

Jika Anda hanya ingin menambahkan teks sederhana, Anda mungkin hanya perlu mengikuti langkah ini.

Jika Anda mengetahui cara menggunakan pen tool, Anda dapat mengeklik alur tertentu untuk membuat teks yang sesuai dengan alur.

4

Klik dan tarik kotak teks sebelum mengetik untuk membatasi teks. Jika Anda perlu membuat teks yang muat pada daerah tertentu, Anda dapat mengeklik dan menarik daerah teks sebelum mulai mengetik. Teks apa saja yang melebihi kotak teks tidak akan tampil, kecuali Anda memperkecil huruf.

5

Klik bagian luar kotak teks atau tekan Ctrl (Control) dan Enter secara bersamaan untuk melihat bagaimana teks Anda terlihat di gambar Photoshop. Jika kotak teks baru terus tampil saat Anda mengeklik, pilih alat lain untuk menutup editor teks. Anda dapat mengeklik ganda teks atau mengeklik teks setelah memilih "Text tool" untuk mengubah jenis huruf dan menyunting teks kapan saja.

Anda tidak dapat menyunting teks setelah memilih opsi "Rasterize". Jika Anda tidak sengaja memilih opsi tersebut, abaikan saja.

Jika layer teks terpilih, tekan Ctrl+T atau Cmd+T untuk mengubah ukuran teks secara manual, alih-alih mengubah ukuran huruf.[2]

Metode 2 dari 2:

Membuat Efek Teks yang Lebih Canggih

1

Klik dan tahan ikon teks berbentuk huruf T di bilah alat untuk mencoba opsi pengetikan teks lain. Klik ikon, kemudian tahan tetikus untuk menampilkan opsi penyuntingan teks alternatif.

Horizontal Type Tool: Opsi ini paling sering digunakan, sekaligus merupakan opsi bawaan. Anda dapat menggunakan opsi ini untuk mengetik huruf secara horizontal dari kiri ke kanan.

Vertical Type Tool: Opsi ini memungkinkan Anda mengetik huruf dari atas ke bawah, alih-alih dari kiri ke kanan.

Horizontal Type Mask Tool: Opsi ini akan mengubah teks Anda menjadi mask, yang dapat digunakan untuk mendapat efek canggih Photoshop. Secara bawaan, opsi ini akan memetakan layer di bawah teks dan menggunakannya untuk "mewarnai" teks.

Vertical Type Mask Tool: Cara kerja alat ini sama dengan Horizontal Type Mask, tetapi memungkinkan Anda mengetik
