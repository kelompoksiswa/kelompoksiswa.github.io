19. cara belajar grammar causative​


Jawaban:

Cara belajar grammar causative​

Caranya dengan menggunakan singkatan ini:

HaMaHeLeG.

Have, Make, Help. Let, get. Artinya menyuruh, membantu, membiarkan.

Rumus Kalimat aktif:

Subject + HaMaHeLeG + someone + verb-1 + object. Artinya Subject menyuruh seseorang untuk melakukan sesuatu.

Contoh:

I have/make/let you write this material. Sama saja dengan mengatakan: I ask you to write this material. Artinya: Saya menyurhmu untuk menulis materi ini.

Rumus Kalimat Pasif:

Subject + HaMaHeLeG + something + verb-3 + by + object. Artinya Subject menyuruh sesusatu untuk dilakukan oleh seseorang.

Contoh:

I have/make/let this material written by you. Sama saja dengan mengatakan: I ask this material written by you. Artinya: Saya menyur materi ini untuk ditulis olehmu.

Dari keseluruhan HaMaHeLeG itu yang beda hanya Get, yaitu sesudah Get harus ada to sesudah someone atau sesudah something.

Contoh:

Kalimat Aktif:

I get you to write this material. Sama saja dengan mengatakan: I ask you to write this material. Artinya: Saya menyurhmu untuk menulis materi ini.

Kalimat Pasif:

Contoh:

I get this material to be written by you. Sama saja dengan mengatakan: I ask this material written by you. Artinya: Saya menyur materi ini untuk ditulis olehmu.

Sedngkan pada kata help, sesudahnya bisa menggunakan to atau  pun tanpa to.

Contoh:

Kalimat Aktif:

I help you write this material. Sama saja dengan mengatakan: I ask you to write this material. Artinya: Saya menyurhmu untuk menulis materi ini.

Atau bisa juga:

I help you to write this material. Sama saja dengan mengatakan: I ask you to write this material. Artinya: Saya menyurhmu untuk menulis materi ini.

Kalimat Pasif:

I help this material written by you. Sama saja dengan mengatakan: I ask this material written by you. Artinya: Saya menyur materi ini untuk ditulis olehmu.

Aatau juga bisa:

I help this material to be written by you. Sama saja dengan mengatakan: I ask this material written by you. Artinya: Saya menyur materi ini untuk ditulis olehmu.

Nanti kalau ada yang salah, bisa diralat,... Dibetulkan lagi,....;
