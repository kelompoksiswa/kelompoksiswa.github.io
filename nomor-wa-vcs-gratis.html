21. Pada VCS Git, terdapat empat jenis workflow. Sebutkan dan jelaskan disertai gambar workflownya dari masing-masing jenis workflow tersebut.


1. Working Directory

Working directory merupakan tempat dimana kita bisa bebas untuk mengedit file

2. Staging Index / Staging Area

Staging Index / Staging Area tempat dimana kita sudah selesai mengedit file dan siap disimpan di local repository

3. Local Repository

Local Repository tempat dimana seluruh file yang sudah kita kita kerjakan, posisi nya masih ada di laptop atau pc masing-masing user

4. Remote Repository

Remote Repository tempat dimana kita menyimpan file yang sudah kita kerjakan, bedanya dengan local repository adalah remote sendiri posisinya ada di server
