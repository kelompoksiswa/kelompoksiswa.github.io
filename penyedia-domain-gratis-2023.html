4. Daftar situs yang tidak menyediakan fasilitas penyewaan domain secara gratis, supaya nama URL blog yang telah dibuat lebih familiar namanya adalah... a. Dot.tk b. My.ga c.  Free.xcv d. Codotvu.co e. Freenom.com​


URL pada blog yang sudah dibuat lebih familiar namanya ialah menggunakan a. Dot.tk. Hal ini dikarenakan pada situs ini tidak menyediakan domain gratis.

Pembahasan:

Nama domain merupakan nama unik buat mengidentifikasi alamat IP yang berupa kombinasi nomor  yg rumit. Dengan nama domain ini, user akan lebih gampang buat mengingat alamat web Anda. Domain name jua tak jarang diklaim menjadi alamat website. Domain merupakan nama unik yang diberikan buat mengidentifikasi alamat (IP address) server personal komputer  misalnya web server atau email server pada internet. Selain mempermudah mengakses website, fungsi domain merupakan sebagai bukti diri berdasarkan sebuah server website agar pengguna nir keliru alamat ketika mengunjunginya. Jadi, fungsi berdasarkan domain merupakan menaruh bukti diri pada suatu website supaya nir terjadi duplikasi serta sifatnya niscaya unique. Oleh lantaran itu, tujuan penggunaan hosting merupakan buat menyimpan keterangan serta data milik website.
